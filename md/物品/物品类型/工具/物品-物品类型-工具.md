# 工具

> 百科 Key: `物品-物品类型-工具`
> 层级: 物品 / 物品类型 / 工具

在进行<span style="color:#6fb6ff">[制造加工](../../制造加工/制造建筑/物品-制造加工-制造建筑.md)</span>时，可以使用对应的工具增加人物的技艺造诣，以达到造诣需求。工具的品级越高，增加的造诣越多。

使用工具进行制造加工，可能会损耗工具的耐久。工具品级相较于引子或物品的品级越高，耐久越不容易损耗。

制造加工时，也可以不使用工具，徒手进行制造加工。徒手制造加工时，人物的造诣将会减半。


工具的获取方式：

- 商会：各大商会出售不同类型的工具。详见「<span style="color:#6fb6ff">[商会](../../../世界/商会/商会/世界-商会-商会.md)</span>」条目。

- 互动：通过互动「<span style="color:#6fb6ff">[换取工具](../../../交互/人物互动/身份互动/交互-人物互动-身份互动.md#换取工具)</span>」，可以换取满耐久的工具。

工具消耗耐久公式：

制造加工时，根据工作难度决定每次消耗的工具耐久：

- <span style="color:#6fb6ff">[制造](../../制造加工/制造/物品-制造加工-制造.md)</span>时：以所用引子的品级为工作难度。

- <span style="color:#6fb6ff">[修理](../../制造加工/修理/物品-制造加工-修理.md)</span>、<span style="color:#6fb6ff">[拆解](../../制造加工/拆解/物品-制造加工-拆解.md)</span>时：以目标物品的品级为工作难度。

- <span style="color:#6fb6ff">[精制](../../制造加工/精制/物品-制造加工-精制.md)</span>时：以所用精制材料的品级为工作难度。

- <span style="color:#6fb6ff">[淬毒](../../制造加工/淬毒/物品-制造加工-淬毒.md)</span>、<span style="color:#6fb6ff">[解毒](../../制造加工/解毒/物品-制造加工-解毒.md)</span>时：以毒药的品级为工作难度。

- <span style="color:#6fb6ff">[改制](../../制造加工/改制/物品-制造加工-改制.md)</span>时：以目标款式的品级为工作难度。
