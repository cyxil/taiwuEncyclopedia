# 制造

> 百科 Key: `物品-制造加工-制造`
> 层级: 物品 / 制造加工 / 制造

通过制造，可以凭借人物自身的技艺产出各类物品。

- 制造必须在对应的<span style="color:#6fb6ff">[制造建筑](../制造建筑/物品-制造加工-制造建筑.md)</span>中进行。

- 制造需要准备对应的<span style="color:#6fb6ff">[工具](../../物品类型/工具/物品-物品类型-工具.md)</span>、<span style="color:#6fb6ff">[引子](../../物品类型/材料/物品-物品类型-材料.md#引子)</span>和<span style="color:#6fb6ff">[资源](../../物品与资源/资源/物品-物品与资源-资源.md#资源)</span>。

- 制造需要人物具备对应的技艺造诣。


### 制造消耗

每次制造需要消耗1个对应材质的<span style="color:#6fb6ff">[引子](../../物品类型/材料/物品-物品类型-材料.md#引子)</span>和一定数量的资源。

选用的引子品级越高，则消耗的资源越多。

制造不同类型物品的资源消耗也各不相同。


制造兵器、护具时，可以选择投入不同比例的金铁、木材、玉石、织物，以改变产物的属性。

制造<span style="color:#6fb6ff">[兵器](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>时，投入的木材越多，兵器的<span style="color:#6fb6ff">[坚韧](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>越高；金铁越多，兵器的破体、破气越高；玉石越多，兵器的<span style="color:#6fb6ff">[破甲](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>越高；织物越多，兵器的重量越轻。

制造<span style="color:#6fb6ff">[护具](../../物品类型/武具/物品-物品类型-武具.md#护具)</span>时，投入的木材越多，护具的<span style="color:#6fb6ff">[坚韧](../../物品类型/武具/物品-物品类型-武具.md#护具)</span>越高；金铁越多，护具的御体、御气越高；玉石越多，护具的<span style="color:#6fb6ff">[破刃](../../物品类型/武具/物品-物品类型-武具.md#护具)</span>越高；织物越多，护具的重量越轻。


### 造诣需求

制造需要人物具备对应的技艺造诣。如果人物的造诣未达到需求，则无法进行制造。

制造选用的引子品级越高，则造诣需求越高。

在制造时使用<span style="color:#6fb6ff">[工具](../../物品类型/工具/物品-物品类型-工具.md)</span>，可以增加人物的造诣。如果不使用工具，徒手进行制造，将导致人物的造诣减半。

**制造造诣需求一览**

| 引子品级 | 造诣需求 |
| --- | --- | 
| 二品 | 360 |
| 三品 | 280 |
| 四品 | 210 |
| 五品 | 150 |
| 六品 | 100 |
| 七品 | 60 |
| 八品 | 30 |
| 九品 | 10 |


### 制造产物

制造产物的材质和品级由选用的引子决定。

引子的品级越高，制造产物的品级越高。


制造产物的基础品级：

- 如果选用金铁、木材、玉石、织物、毒物引子：产物基础品级 = 引子品级 - 1。

- 如果选用食材引子：产物基础品级 = 引子品级 + 已掌握的厨艺技艺数量，最高为三品。

  制造食物时，造诣需求为产物基础品级对应的造诣需求。

- 如果制造丹药，则产物基础品级为：

  <span style="color:#DCDEE0FF">八品</span>药材引子：奇方产物为<span style="color:#80817FFF">九品</span>；正方产物为<span style="color:#5D92C3FF">六品</span>。

  <span style="color:#5D92C3FF">六品</span>药材引子：奇方产物为<span style="color:#DCDEE0FF">八品</span>；正方产物为<span style="color:#509296FF">五品</span>。

  <span style="color:#885390FF">四品</span>药材引子：奇方产物为<span style="color:#9EB767FF">七品</span>；正方产物为<span style="color:#885390FF">四品</span>。

  <span style="color:#CE5D20FF">二品</span>药材引子：奇方产物为<span style="color:#5D92C3FF">六品</span>；正方产物为<span style="color:#BE8A2FFF">三品</span>。

  制造丹药时如果指定字诀，则造诣需求也会变化为相应的品级。


制造产物的最终品级：

- 如果人物的造诣达到造诣需求的150%，则制造产物的最终品级 = 基础品级 + 1。

- 如果人物的造诣达到造诣需求的200%，且制造建筑周围建造了提升产物品级的扩展建筑，则制造产物的最终品级 = 基础品级 + 2。

能够提升制造产物品级、降低造诣需求的扩展建筑，详见「<span style="color:#6fb6ff">[制造建筑](../制造建筑/物品-制造加工-制造建筑.md)</span>」条目的「制造建筑一览」表格。


### 字诀

部分类型的物品在制造时可以指定字诀。不同字诀的制造产物具有不同的效果倾向。

制造时，如果指定字诀，将会额外增加制造的造诣需求（丹药的「奇」字诀除外）。

如果不指定字诀，则制造产物为随机一种字诀对应的产物。


指定字诀公式：

增加造诣需求 = （引子品级 + 1） × 15


### 精益求精

制造兵器、护具或宝物时采用精益求精，可以使制造出来的装备必定具有一项随机的<span style="color:#6fb6ff">[装备特性](../../物品类型/装备/物品-物品类型-装备.md#装备特性)</span>，但此次制造的造诣需求也会提高50%。
