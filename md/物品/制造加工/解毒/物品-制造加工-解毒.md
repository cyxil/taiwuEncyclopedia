# 解毒

> 百科 Key: `物品-制造加工-解毒`
> 层级: 物品 / 制造加工 / 解毒

通过解毒，可以解除物品中淬有的毒素。


解毒的需求和消耗：

- 解毒必须在<span style="color:#9eb767">幽室</span>中进行。

- 解毒需要消耗解毒<span style="color:#6fb6ff">[丹药](../../物品类型/药毒/物品-物品类型-药毒.md#丹药)</span>。解毒丹药必须与物品所淬的毒素类型对应，且丹药品级不低于毒素的品级。

- 解毒需要人物具备对应的毒术造诣。

  物品所淬的毒素品级越高，解毒的造诣需求越高。

  如果人物的毒术造诣未达到需求，则无法解毒。

  在解毒时使用<span style="color:#6fb6ff">[工具](../../物品类型/工具/物品-物品类型-工具.md)</span>，可以增加人物的造诣。如果不使用工具，徒手进行解毒，将导致人物的造诣减半。


<span style="color:#6fb6ff">[药霜](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>、<span style="color:#6fb6ff">[毒砂](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>本身所含的毒素不能解除。


### 提取毒素

解毒时，可以提取出去除的毒素，转化为对应的毒药，但此次解毒的毒术造诣需求会提高50%。
