# 制造建筑

> 百科 Key: `物品-制造加工-制造建筑`
> 层级: 物品 / 制造加工 / 制造建筑

制造建筑是可以在其中制造加工各类物品的建筑。

**制造建筑一览**

| 制造建筑 | 造诣需求 | 建筑功能 | 提高产物品级 | 降低造诣需求 |
| --- | --- | --- | --- | --- | 
| <span style="color:#9eb767">火炼室</span> | 锻造 | 可以制造、修理或精制金铁材质的兵器、护具、宝物。 | <span style="color:#9eb767">龙泉七星井</span> | <span style="color:#9eb767">神火铸</span> |
| <span style="color:#9eb767">木工房</span> | 制木 | 可以制造、修理或精制木材材质的兵器、护具、宝物；可以制造舟车。 | <span style="color:#9eb767">幽冥窑</span> | <span style="color:#9eb767">神木林</span> |
| <span style="color:#9eb767">巧匠屋</span> | 巧匠 | 可以制造、修理或精制玉石材质的兵器、护具、宝物。 | <span style="color:#9eb767">玲珑台</span> | <span style="color:#9eb767">神光璧</span> |
| <span style="color:#9eb767">绣楼</span> | 织锦 | 可以制造、修理或精制织物材质的兵器、护具、宝物；可以制造衣装或绳索；可以改制衣装。 | <span style="color:#9eb767">彩桑园</span> | <span style="color:#9eb767">神彩梭</span> |
| <span style="color:#9eb767">药房</span> | 医术 | 可以制造丹药、药霜；可以修理药霜。 | <span style="color:#9eb767">水火青炉</span> | <span style="color:#9eb767">神农涧</span> |
| <span style="color:#9eb767">幽室</span> | 毒术 | 可以制造毒药、毒砂；可以修理毒砂；可以给物品淬毒、解毒。 | <span style="color:#9eb767">血池</span> | <span style="color:#9eb767">神龙柱</span> |
| <span style="color:#9eb767">食窖</span> | 厨艺 | 可以制造菜肴。 | <span style="color:#9eb767">香料馆</span> | <span style="color:#9eb767">龙宫</span> |

当太吾位于<span style="color:#6fb6ff">[定居点](../../../世界/地图/地格/世界-地图-地格.md#定居点)</span>地格时，可以使用太吾村或当地的制造建筑进行制造。

当太吾不位于定居点地格时，也可以使用太吾村的制造建筑进行制造，但只能使用仓库中的引子和工具，制造产物也只能存入仓库。

在制造建筑的2格范围内建造相应的扩展建筑，能够提升制造产物的品级，或降低制造加工的造诣需求。
