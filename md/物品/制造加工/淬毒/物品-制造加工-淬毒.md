# 淬毒

> 百科 Key: `物品-制造加工-淬毒`
> 层级: 物品 / 制造加工 / 淬毒

通过淬毒，可以消耗<span style="color:#6fb6ff">[毒药](../../物品类型/药毒/物品-物品类型-药毒.md#毒药)</span>，为各类物品添加<span style="color:#6fb6ff">[毒素](../../../人物/伤病/毒素/人物-伤病-毒素.md)</span>。

每件物品最多淬有3种不同类型的毒素。


不同类型的淬毒物品的用途：

- 兵器：使用淬毒的兵器攻击时，可以向敌人施加毒素。

  <span style="color:#6fb6ff">[药霜](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>、<span style="color:#6fb6ff">[毒砂](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>本身所含的毒素不会影响淬毒。

- 护具：穿戴着淬毒护具的部位受到攻击时，可以向敌人施加毒素。

- 宝物：装备淬毒的宝物，可以在战斗中降低敌人的<span style="color:#6fb6ff">[毒素抵抗](../../../人物/伤病/毒素/人物-伤病-毒素.md#毒素抵抗)</span>。

- 食物、丹药：当人物服食淬毒的食物、丹药时，将会被施加对应的毒素。

  食物、丹药淬毒后，还可以延长其服食效果的持续时间。详见「<span style="color:#6fb6ff">[服食效果](../../../人物/伤病/服食汲饮/人物-伤病-服食汲饮.md#服食效果)</span>」条目。

- 书籍：当人物研读淬毒的书籍时，将会被施加对应的毒素。


淬毒的限制：

- 可以淬毒的物品有：<span style="color:#6fb6ff">[食物](../../物品类型/食物/物品-物品类型-食物.md)</span>、<span style="color:#6fb6ff">[丹药](../../物品类型/药毒/物品-物品类型-药毒.md#丹药)</span>、<span style="color:#6fb6ff">[兵器](../../物品类型/武具/物品-物品类型-武具.md#兵器)</span>、<span style="color:#6fb6ff">[护具](../../物品类型/武具/物品-物品类型-武具.md#护具)</span>、<span style="color:#6fb6ff">[宝物](../../物品类型/武具/物品-物品类型-武具.md#宝物)</span>、<span style="color:#6fb6ff">[书籍](../../物品类型/书籍/物品-物品类型-书籍.md)</span>。

- 当物品上淬有未知毒素时，需要先<span style="color:#6fb6ff">[验毒](../验毒/物品-制造加工-验毒.md)</span>，然后才能淬毒。

- 当物品上已淬有毒素时，将无法使用与这些毒素类型相同，且品级相同或更低的毒药进行淬毒。

- 当物品上已淬有3种不同类型的毒素时，将无法使用与这些毒素类型均不同的毒药进行淬毒。但可以使用相同类型、品级更高的毒药，淬毒后将替换物品原有的同种毒素。


淬毒的需求和消耗：

- 淬毒必须在<span style="color:#9eb767">幽室</span>中进行。

- 淬毒需要消耗毒药。

- 淬毒需要人物具备对应的毒术造诣。

  选用的毒药品级越高，淬毒的造诣需求越高。

  如果人物的毒术造诣未达到需求，则无法淬毒。

  在淬毒时使用工具，可以增加人物的造诣。如果不使用工具，徒手进行淬毒，将导致人物的造诣减半。


### 凝炼淬毒

淬毒时，使用<span style="color:#6fb6ff">[毒素类型](../../../人物/伤病/毒素/人物-伤病-毒素.md)</span>和<span style="color:#6fb6ff">[毒素等级](../../../人物/伤病/毒素/人物-伤病-毒素.md#毒素等级)</span>均相同的3份毒药进行淬毒，可以使毒素量提高50%，但不会改变毒素等级。

如果采用凝炼淬毒，此次淬毒的毒术造诣需求会提高50%。
