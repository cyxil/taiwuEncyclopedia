# 茶马帮

> 百科 Key: `产业-特殊建筑-茶马帮`
> 层级: 产业 / 特殊建筑 / 茶马帮

茶马帮是向西域各国派遣商队，以换取<span style="color:#6fb6ff">[西域珍宝](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#西域珍宝)</span>的建筑，在主要故事达到一定进度后方可建造。


### 货物

在商队出发前，太吾可以自行选择商队携带的货物。

命令商队出发后，商队便会离开太吾村，向西行进。行进途中，每次月份更替时，商队均会尝试用货物换取西域珍宝。

太吾可以随时命令商队返程，即向东返回太吾村。返程途中，商队将不会再换取西域珍宝。

商队回到太吾村后，太吾可以获取商队换得的西域珍宝，并重新选择商队携带的货物。


茶马帮交换公式：

- 货物价值 = 物品价值 + 茶马帮知名度

- 换取西域珍宝的品级：40%概率获得货物价值对应品级的西域珍宝；60%概率获得低一级的西域珍宝

- 如果货物价值只能够换取<span style="color:#509296FF">五品</span>的西域珍宝，则换得概率% = 货物价值 ÷ 五品西域珍宝价值 × 100 - 50


### 补给

商队行进或返程时，每个月均会消耗补给。

当补给耗尽后，将有可能在月份更替时丢失携带的货物或西域珍宝。

- 商队出发时的补给数量为100；每月消耗5补给。

- 商队行进途中，有可能遇到额外消耗或获取补给的随机事件。


茶马帮丢失货物概率公式：

每月丢失货物概率% = 2^（补给耗尽月数 - 1），最大100%


### 知名度

随着茶马帮商队抵达更远的距离，茶马帮的知名度也会提升。

茶马帮的知名度越高，则商队可携带的货物上限越多，换得西域珍宝的品级越高。

- 茶马帮向西行进时，每个月都会获得100知名度。

- 部分随机事件也可能影响茶马帮的知名度。

- 茶马帮的知名度取决于单次行程所达到的最高知名度。


### 天气

商队在行进或返程途中会经过不同的地形，并可能遇到各种恶劣天气，导致商队额外消耗补给，或增加丢失货物的概率。

### 事件

商队行进或返程途中，每个月有可能遇到各类事件，对商队产生不同的影响。

**茶马帮事件一览**

| 事件 | 效果 |
| --- | --- | 
| 发现村落 | 可以用货物换取补给。可换取的补给总量为60~80。 换取补给 = 5 + （货物品级 + 1） × 5 |
| 路遇商队 | 可以用货物换取补给。可换取的补给总量为40~60。 换取补给 = 5 + （货物品级 + 1） × 5 |
| 发现果林 | 可以命令商队暂时停留，搜寻补给。 可搜寻的补给总量为30~50；每月可搜寻的补给为10~15。 |
| 发现商队残骸 | 随机获得1件五品~三品的货物。仅当补给不为0时可以触发。 |
| 遇到盗贼 | 随机丢失1~3件货物。 |
| 颠簸损坏 | 随机丢失1件货物。 |
| 获得援助 | 随机获得10~30补给。 |
| 偶得野味 | 随机获得10~30补给。 |
| 水土不服 | 随机消耗10~30补给。 |
| 援助路人 | 随机消耗5~10补给，知名度提升100。 |
| 传回见闻 | 获得1条随机的西域见闻，知名度提升20。 |
| 发现动物 | 知名度提升20。 |
| 发现植物 | 知名度提升20。 |
| 遇到罕见天气 | 知名度提升20。 |
| 遇到海市蜃楼 | 知名度提升30。 |
| 发现野人 | 知名度提升50。 |
| 发现聚落 | 知名度提升50。 |
| 迷失道路 | 知名度降低100。 |
