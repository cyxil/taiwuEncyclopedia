# 村民

> 百科 Key: `产业-太吾村民-村民`
> 层级: 产业 / 太吾村民 / 村民

太吾村民是除太吾传人外，所有从属于太吾村的人物。太吾村民均服从太吾的调遣。

- 太吾可以指派<span style="color:#6fb6ff">[成年](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#身龄)</span>的村民参与太吾村的<span style="color:#6fb6ff">[产业建设](../../产业/产业建设/产业-产业-产业建设.md)</span>和<span style="color:#6fb6ff">[经营](../../产业/产业经营/产业-产业-产业经营.md)</span>。

- 太吾可以对成年的村民授予村民身份，赋予各自身份对应的职能。

- 太吾可以无条件地将村民<span style="color:#6fb6ff">[邀为同道](../../../交互/人物互动/亲近/交互-人物互动-亲近.md#邀为同道)</span>。

- 太吾可以在村民名册中，查看所有村民的信息和状态，或将村民逐出太吾村。


村民主要通过以下方式获得：

- 通过太吾村产业中的人才建筑，可以招募特定资质卓越的村民。

- 将非太吾村的人物<span style="color:#6fb6ff">[邀请回村](../../../交互/人物互动/亲近/交互-人物互动-亲近.md#邀请回村)</span>后，此人物便会成为太吾村民。

- 救治<span style="color:#6fb6ff">[失心人](../../../启程/相枢/相枢/启程-相枢-相枢.md#失心人)</span>后，可以使其成为太吾村民。

- 成功劝说<span style="color:#6fb6ff">[俘虏](../../../交互/其他交互/俘虏/交互-其他交互-俘虏.md)</span>加入同道后，此人物便会成为太吾村民。

- 从属于太吾村的人物所生育的后代均为太吾村民。


### 村民身份

太吾可以在太吾村的传承名谱中查看、授予或免除村民身份。

村民身份按品阶由低到高依次为：农户、匠人、大夫、商人、文人、护冢、使者。


任何已<span style="color:#6fb6ff">[成年](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#身龄)</span>的村民都可以被授予任意村民身份。

授予身份需要消耗威望。随着一个身份的村民数量越多，授予此身份的威望消耗越多。


被授予身份的村民将获得一个对应的村民身份特性，免除身份后失去此特性。

被授予身份的村民在月份更替时，会从太吾村的公库中自行取用<span style="color:#6fb6ff">[村民衣装](../../../物品/物品类型/行装/物品-物品类型-行装.md#衣装)</span>，以及造诣需求对应的书籍。

授予村民身份消耗公式：

<span style="color:#6fb6ff">[农户](../农户/产业-太吾村民-农户.md)</span>：消耗威望 = 100×（现有农户人数+1）

<span style="color:#6fb6ff">[匠人](../匠人/产业-太吾村民-匠人.md)</span>：消耗威望 = 250×（现有匠人人数+1）

<span style="color:#6fb6ff">[大夫](../大夫/产业-太吾村民-大夫.md)</span>：消耗威望 = 500×（现有大夫人数+1）

<span style="color:#6fb6ff">[商人](../商人/产业-太吾村民-商人.md)</span>：消耗威望 = 750×（现有商人人数+1）

<span style="color:#6fb6ff">[文人](../文人/产业-太吾村民-文人.md)</span>：消耗威望 = 1000×（现有文人人数+1）

<span style="color:#6fb6ff">[护冢](../护冢/产业-太吾村民-护冢.md)</span>：消耗威望 = 1500×（现有护冢人数+1）

<span style="color:#6fb6ff">[使者](../使者/产业-太吾村民-使者.md)</span>：消耗威望 = 2500×（现有使者人数+1）

- 授予每种身份的第一个消耗为0；第二个消耗减半。


### 身份职能

不同的村民身份有各自的职能，履行职能需要特定的造诣和七元赋性。村民对应的造诣和七元越高，就能够越好地履行身份职能。

每个身份的职能分为以下几个方面：

- 产业经营：在经营建筑中，仅当主事具有对应身份时，才能<span style="color:#6fb6ff">[培养](../../产业/产业经营/产业-产业-产业经营.md#培养)</span>学徒或<span style="color:#6fb6ff">[代制](../../../物品/制造加工/代制/物品-制造加工-代制.md)</span>物品。

- 派遣工作：可以将具有身份的村民派遣至任意已开通驿站的地点，执行相应的工作。

- 自发行为：具有身份的村民在未参与产业经营和派遣工作时（<span style="color:#6fb6ff">[使者](../使者/产业-太吾村民-使者.md)</span>无此限制），会在月份更替时自发执行相应的行为。
