# 比试

> 百科 Key: `交互-人物互动-比试`
> 层级: 交互 / 人物互动 / 比试

通过比试互动，可以与他人进行战斗、较艺或促织决斗。


### 促织决斗

互动内容：与对方进行促织决斗。关于促织决斗的具体规则，详见「<span style="color:#6fb6ff">[促织决斗](../../../游历/促织/促织决斗/游历-促织-促织决斗.md)</span>」条目。

使用需求：对方好感为<span style="color:#239360FF">冷淡</span>及以上；太吾持有至少3只存活的促织

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 若太吾胜利，则赢得对方的押注；若太吾失败，则失去自己的押注。

- 若太吾胜利，则对方好感上升，心情降低；若太吾失败，则对方好感降低，心情上升。


### 较艺比试

互动内容：与对方进行较艺。关于较艺的具体规则，详见「<span style="color:#6fb6ff">[较艺](../../../修习/技艺/较艺/修习-技艺-较艺.md)</span>」条目。

使用需求：对方好感为<span style="color:#1ABAC8FF">热忱</span>及以上

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行较艺。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


### 切磋武功

互动内容：向对方发起点到为止的切磋，双方以<span style="color:#6fb6ff">[切磋](../../../战斗/战斗基础/战斗类型/战斗-战斗基础-战斗类型.md#切磋)</span>的方式进行战斗。

使用需求：对方好感为<span style="color:#1ABAC8FF">热忱</span>及以上

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行切磋。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


### 发起挑战

互动内容：向对方发起真刀真枪的较量，双方以<span style="color:#6fb6ff">[恶斗](../../../战斗/战斗基础/战斗类型/战斗-战斗基础-战斗类型.md#恶斗)</span>的方式进行战斗。

使用需求：对方好感为<span style="color:#1ABAC8FF">热忱</span>及以上；对方<span style="color:#6fb6ff">[精纯](../../../修习/内力/精纯境界/修习-内力-精纯境界.md)</span>不低于太吾

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行恶斗。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


比试互动效果公式：

如果太吾胜利，对方的好感和心情根据其立场变化：

- 对方好感：<span style="color:#FFE78FFF">刚正</span> = +600；<span style="color:#9FE0DCFF">仁善</span> = +1200；<span style="color:#FFFFFFFF">中庸</span> = +600；<span style="color:#B975FFFF">叛逆</span> = +1200；<span style="color:#C6272EFF">唯我</span> = +600

- 对方心情：<span style="color:#FFE78FFF">刚正</span> = -2；<span style="color:#9FE0DCFF">仁善</span> = -3；<span style="color:#FFFFFFFF">中庸</span> = -2；<span style="color:#B975FFFF">叛逆</span> = -3；<span style="color:#C6272EFF">唯我</span> = -2

如果太吾失败，对方的好感和心情根据其立场变化：

- 对方好感：<span style="color:#FFE78FFF">刚正</span> = -1200；<span style="color:#9FE0DCFF">仁善</span> = -600；<span style="color:#FFFFFFFF">中庸</span> = 0；<span style="color:#B975FFFF">叛逆</span> = -1200；<span style="color:#C6272EFF">唯我</span> = -600

- 对方心情：<span style="color:#FFE78FFF">刚正</span> = +2；<span style="color:#9FE0DCFF">仁善</span> = +3；<span style="color:#FFFFFFFF">中庸</span> = +2；<span style="color:#B975FFFF">叛逆</span> = +3；<span style="color:#C6272EFF">唯我</span> = +2

如果太吾在切磋、挑战中逃跑，且对方立场为<span style="color:#FFE78FFF">刚正</span>，则对方好感额外-1000。
