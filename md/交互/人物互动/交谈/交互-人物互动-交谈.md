# 交谈

> 百科 Key: `交互-人物互动-交谈`
> 层级: 交互 / 人物互动 / 交谈

通过交谈互动，可以提升或降低他人的<span style="color:#6fb6ff">[好感](../../../人物/人物信息/好感/人物-人物信息-好感.md)</span>，或获取<span style="color:#6fb6ff">[见闻](../../交互信息/见闻/交互-交互信息-见闻.md)</span>、<span style="color:#6fb6ff">[秘闻](../../交互信息/秘闻/交互-交互信息-秘闻.md)</span>。


### 见闻闲谈

互动内容：通过与对方谈论见闻，增加对方的好感，减少对方的戒心。根据所谈论的见闻，还可能产生不同的额外效果。

使用需求：太吾持有可使用的见闻

使用限制：每月对同一人物限用一次

消耗精力：1

互动效果：

- 增加对方的好感。交谈效果越好，增加的好感越多。

- 减少对方的戒心。见闻等级越高，交谈效果越好，减少的戒心越多。

- 谈论不同类型的见闻，还可能造成不同的额外效果。

- 与不同的人物谈论见闻，可能会有不同的回应，分为显著、普通、无效三种。若交谈无效，则不会产生任何效果。

各类型见闻的具体交谈效果，详见「<span style="color:#6fb6ff">[见闻](../../交互信息/见闻/交互-交互信息-见闻.md)</span>」条目。


见闻闲谈增加好感公式：

人物好感增加 = 基础好感公式 × 对应见闻闲谈效果倍率

- 基础好感公式 = 600 + 太吾魅力值 ÷ 3 + 太吾名誉值 × 3

- 效果倍率：不同的见闻倍率不同，详见「<span style="color:#6fb6ff">[见闻](../../交互信息/见闻/交互-交互信息-见闻.md)</span>」条目。


见闻闲谈减少戒心公式：

人物戒心减少 = （见闻等级 + 1） × 效果倍率 × 2000

- 效果倍率：显著 = 150%；普通 = 100%


### 使用秘闻

互动内容：与对方谈论<span style="color:#6fb6ff">[秘闻](../../交互信息/秘闻/交互-交互信息-秘闻.md)</span>。

消耗精力：1

消耗威望：由所传播的秘闻决定

互动效果：

- 将太吾所知晓的秘闻传播给对方。

- 根据对方与此秘闻的关联程度，可能引发不同的后续互动。

- 当秘闻的知晓人数达到10人后，此秘闻将公开，并可能对关联人物和传播来源造成影响。


### 打探见闻

互动内容：打探对方所持有的<span style="color:#6fb6ff">[见闻](../../交互信息/见闻/交互-交互信息-见闻.md)</span>。

使用限制：对方好感为<span style="color:#B975FFFF">喜爱</span>及以上

消耗精力：10

互动效果：

- 获得一条对方持有，且太吾未持有的见闻。

- 仅能打探地方见闻、门派见闻和志向见闻。

### 赞颂夸奖

互动内容：通过称赞对方，增加对方的好感。

使用限制：与羞辱指责每月对同一人物共限用一次

消耗精力：1

互动效果：

- 称赞赋性：根据对方的数值为30及以上的七元赋性数量（除福缘、合道外）增加好感。若对方的赋性值均低于30，则互动无效。

- 称赞魅力：根据对方的魅力增加好感。若对方的魅力为出众及以下，则互动无效。

- 称赞名誉：根据对方的名誉增加好感。若对方的名誉为默默无闻及以下，则互动无效。

- 称赞特性：根据对方的正面通常特性的数量增加好感。若对方没有正面通常特性，则互动无效。

- 称赞财富：根据对方的银钱超出太吾的倍率增加好感。若对方的银钱少于太吾，则互动无效。

对于不同立场的人物，互动效果不同。如果对方立场为<span style="color:#B975FFFF">叛逆</span>，赞颂夸奖反而会导致其好感降低。


赞颂夸奖增加好感公式：

对方好感增加 =  Min（称赞等级，称赞等级上限） × 每级好感 × 对方立场影响

- 称赞等级：

  赋性：对方的数值为30及以上的赋性数量（除福缘、合道外）

  魅力：对方魅力等级-4

  名誉：根据对方的名誉等级：立身扬名 = 1；声驰千里 = 2；誉满天下 = 3

  特性：对方的正面通常特性数量

  财富：（对方银钱 - 太吾银钱） ÷ Max（1，太吾银钱 ÷ 5）

- 称赞等级上限：赋性 = 5；魅力 = 4；名誉 = 3；特性 = 5；财富 = 5

- 每级好感：赋性 = 120；魅力 = 150；名誉 = 200；特性 = 120；财富 = 120

- 对方立场影响：<span style="color:#FFE78FFF">刚正</span> = 2；<span style="color:#9FE0DCFF">仁善</span> = 3；<span style="color:#FFFFFFFF">中庸</span> = 4；<span style="color:#B975FFFF">叛逆</span> = -2；<span style="color:#C6272EFF">唯我</span> = 3

夸奖成功时，根据对方立场改变其戒心：<span style="color:#FFE78FFF">刚正</span> = -5000；<span style="color:#9FE0DCFF">仁善</span> = -10000；<span style="color:#FFFFFFFF">中庸</span> = -2000；<span style="color:#B975FFFF">叛逆</span> = +5000；<span style="color:#C6272EFF">唯我</span> = -5000


### 羞辱指责

互动内容：通过指责对方，降低对方的好感。

使用限制：与赞颂夸奖每月对同一人物共限用一次

消耗精力：1

互动效果：

- 指责赋性：根据对方的数值为10以下的七元赋性数量（除福缘、合道外）降低好感。若对方的赋性均不小于10，则互动无效，太吾获得名誉「自取其辱」。

- 指责魅力：根据对方的魅力降低好感。若对方的魅力为出众及以上，则互动无效，太吾获得名誉「自取其辱」。

- 指责名誉：根据对方的名誉降低好感。若对方的名誉为默默无闻及以上，则互动无效，太吾获得名誉「自取其辱」。

- 指责特性：根据对方的负面通常特性的数量降低好感。若对方没有负面通常特性，则互动无效，太吾获得名誉「自取其辱」。

- 指责财富：根据太吾的银钱超出对方的倍率降低好感。若对方的银钱多于太吾，则互动无效，太吾获得名誉「自取其辱」。

对于不同立场的人物，互动效果不同。如果对方立场为<span style="color:#B975FFFF">叛逆</span>，羞辱指责反而会导致其好感上升。

如果对方的好感过低，将可能与太吾结下仇怨。


羞辱指责效果公式：

对方好感减少 =  Min（指责等级，指责等级上限） × 每级好感 × 对方立场影响

- 指责等级：

  赋性：对方的数值为10及以下的赋性数量（除福缘、合道外）

  魅力：4-对方魅力等级

  名誉：根据对方的名誉等级：名声败坏 = 1；千夫所指 = 2；妖魔鬼怪 = 3

  特性：对方的负面通常特性数量

  财富：（太吾银钱 - 对方银钱） ÷ Max（1，对方银钱 ÷ 5）

- 指责等级上限：赋性 = 5；魅力 = 4；名誉 = 3；特性 = 5；财富 = 5

- 每级好感：赋性 = 120；魅力 = 150；名誉 = 200；特性 = 120；财富 = 120

- 对方立场影响：<span style="color:#FFE78FFF">刚正</span> = 2；<span style="color:#9FE0DCFF">仁善</span> = 3；<span style="color:#FFFFFFFF">中庸</span> = 4；<span style="color:#B975FFFF">叛逆</span> = -2；<span style="color:#C6272EFF">唯我</span> = 3

指责成功时，根据对方立场改变其戒心：<span style="color:#FFE78FFF">刚正</span> = +5000；<span style="color:#9FE0DCFF">仁善</span> = +5000；<span style="color:#FFFFFFFF">中庸</span> = +2000；<span style="color:#B975FFFF">叛逆</span> = -10000；<span style="color:#C6272EFF">唯我</span> = +5000


根据对方的立场和好感，判定是否与太吾结仇：

- 对方立场为<span style="color:#FFE78FFF">刚正</span>：好感不高于敌视时结仇

- 对方立场为<span style="color:#9FE0DCFF">仁善</span>：好感不高于仇视时结仇

- 对方立场为<span style="color:#FFFFFFFF">中庸</span>：好感不高于憎恨时结仇

- 对方立场为<span style="color:#B975FFFF">叛逆</span>：好感不高于鄙视时结仇

- 对方立场为<span style="color:#C6272EFF">唯我</span>：好感不高于陌路时结仇


### 赠送礼物

互动内容：通过将物品赠予对方，增加对方的好感。

消耗精力：3

互动效果：

- 根据赠送的物品，对方增加相应的好感和心情，并减少戒心。

- 如果物品的礼物品级低于对方的身份品阶，则无法赠送此物品。

- 如果赠送食物，对方接受礼物后将直接服食。若对方此时没有可用的服食栏位，则退回礼物。如果是触犯对方所属<span style="color:#6fb6ff">[门派戒律](../../../门派/门派概述/门派/门派-门派概述-门派.md)</span>的食物，则无法赠送。

- 如果赠送<span style="color:#6fb6ff">[淬毒](../../../物品/制造加工/淬毒/物品-制造加工-淬毒.md)</span>的物品，根据对方的毒术造诣，有可能被对方察觉。


启用武师志向技能「<span style="color:#6fb6ff">[江湖中人](../../../游历/志向/武师/游历-志向-武师.md#江湖中人)</span>」、平民志向技能「<span style="color:#6fb6ff">[乡亲父老](../../../游历/志向/平民/游历-志向-平民.md#乡亲父老)</span>」，可以提升赠送礼物的效果。


赠送礼物效果公式：

对方好感增加 = 物品的赠礼好感 × 耐久影响 × 喜恶影响 × 淬毒装备影响 × 志向技能影响

对方心情增加 = 物品的赠礼心情 × 耐久影响 × 喜恶影响

- 耐久影响% = 25 + 当前耐久 ÷ 耐久上限 × 75

- 喜恶影响：对方喜好物品 = 200%；对方厌恶物品 = 50%

- 淬毒装备影响：如果赠送淬毒的装备并被对方察觉，且对方门派戒律禁止用毒，则最终增加的好感 × 50%

- 志向技能影响：详见「<span style="color:#6fb6ff">[江湖中人](../../../游历/志向/武师/游历-志向-武师.md#江湖中人)</span>」、「<span style="color:#6fb6ff">[乡亲父老](../../../游历/志向/平民/游历-志向-平民.md#乡亲父老)</span>」词条

对方戒心减少 = 物品基础赠送好感 × 喜恶影响 × 10

- 喜恶影响：对方喜好物品 = 120%；对方厌恶物品 = 80%


如果赠送淬毒的物品：

- 判断对方能否察觉：

  如果对方的毒术造诣 ≥ 所淬毒素中最低品级的解毒需求，则对方察觉

  九品~一品毒素的解毒造诣需求分别为10/30/60/100/150/210/280/360/450

- 赠送淬毒的装备并被对方察觉时：如果对方门派戒律禁止用毒，则最终增加的好感减半；否则对方照常收下礼物。

- 赠送其他淬毒的物品并被对方察觉时：视为太吾毒害失败，对方好感-6000，太吾心情-5。


如果赠送奇书：

- 根据对方的立场和战力排名，满足条件时将收下奇书，否则退回：

  <span style="color:#FFE78FFF">刚正</span> = 前5名；<span style="color:#9FE0DCFF">仁善</span> = 前25名；<span style="color:#FFFFFFFF">中庸</span> = 前100名；<span style="color:#B975FFFF">叛逆</span> = 前1000名；<span style="color:#C6272EFF">唯我</span> = 前500名
