# 俘虏

> 百科 Key: `交互-其他交互-俘虏`
> 层级: 交互 / 其他交互 / 俘虏

人物在战斗中可以使用绳索<span style="color:#6fb6ff">[抓捕](../../../战斗/指令与行为/战斗行为/战斗-指令与行为-战斗行为.md#使用物品)</span>敌人，使其成为自己的俘虏。


### 关押栏位

俘虏会占用人物的关押栏位。

若俘虏的数量超出关押栏位上限，将会增加俘虏逃跑的概率。

人物的初始关押栏位为1，可以通过装备<span style="color:#6fb6ff">[代步](../../../物品/物品类型/行装/物品-物品类型-行装.md#代步)</span>增加关押栏位。


### 抵抗意志

俘虏被抓捕并关押后，会对关押者产生一定的抵抗意志。

月份更替时，若俘虏的抵抗意志达到100，俘虏将尝试逃跑。俘虏的抵抗意志越高，绑缚的绳索品级越低，超出关押栏位的俘虏数量越多，则俘虏逃跑越容易成功。

可以随时更换绑缚的绳索，使用过的绳索将被丢弃。


每次月份更替时，俘虏的抵抗意志会发生变化。

俘虏的立场越趋近于<span style="color:#C6272EFF">唯我</span>、<span style="color:#6fb6ff">[精纯](../../../修习/内力/精纯境界/修习-内力-精纯境界.md)</span>相较于关押者越高、对关押者好感越低，则抵抗意志越容易上升；反之，抵抗意志越容易下降。


俘虏逃跑成功率公式：

逃跑成功率% = （抵抗意志 - 100） × （100 - 绳索影响） + 超出关押栏位的俘虏数量 × 20

- 绳索影响：九品~一品的绳索 = 20/25/30/35/40/45/50/60/70


抵抗意志变化公式：

每月抵抗意志变化 = 俘虏立场影响 × 10 + 精纯影响 - 好感影响

- 俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 1；<span style="color:#9FE0DCFF">仁善</span> = 2；<span style="color:#FFFFFFFF">中庸</span> = 3；<span style="color:#B975FFFF">叛逆</span> = 4；<span style="color:#C6272EFF">唯我</span> = 5

- 精纯影响 = 18 + 俘虏精纯 - 关押者精纯

- 好感影响 = 俘虏对关押者的好感 ÷ 500


### 俘虏互动

俘虏不能进行通常的人物互动，仅能进行俘虏相关的互动。

通过俘虏互动，可以尝试劝说俘虏加入同道，或予以惩罚。


太吾可以与俘虏进行下列的互动：

- 释放：释放此俘虏。释放俘虏后，绑缚的绳索将收回到行囊。

- 服食：使俘虏服食太吾持有的食物、丹药或毒药。

  根据服食的物品和俘虏的立场，可能会改变俘虏的抵抗意志和好感。

- 给予：将太吾持有的物品赠予俘虏。

  根据赠予的物品和俘虏的立场，可能会改变俘虏的抵抗意志和好感。

- 劝说：劝说俘虏加入同道。

  劝说成功后，将降低俘虏的抵抗意志。若抵抗意志为0及以下，且对方身份品阶不高于<span style="color:#6fb6ff">[侵袭进度](../../../启程/相枢/相枢/启程-相枢-相枢.md#侵袭进度)</span>的同道品阶限制，则俘虏将会加入太吾的同道。

- 转移：将自己或同道所关押的俘虏转交给队伍中的其他人物。

- 处罚：与俘虏进行恶斗。

  若太吾战胜，将增加俘虏的抵抗意志；否则俘虏将会逃跑。

- 没收：没收俘虏持有的物品、资源或银钱。

  将会增加俘虏的抵抗意志，并产生恩义累积。

- 询问：询问俘虏持有的见闻或秘闻。

  需要俘虏的抵抗意志较低时才能成功。不同立场的俘虏要求的抵抗意志不同。


俘虏互动相关公式：

服食：

- 如果为<span style="color:#6fb6ff">[毒药](../../../物品/物品类型/药毒/物品-物品类型-药毒.md#毒药)</span>或<span style="color:#6fb6ff">[淬毒](../../../物品/制造加工/淬毒/物品-制造加工-淬毒.md)</span>食物，并被俘虏察觉：俘虏好感-3000，抵抗意志增加 = 物品品级 × 俘虏立场影响

  俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 0.5；<span style="color:#9FE0DCFF">仁善</span> = 1；<span style="color:#FFFFFFFF">中庸</span> = 0；<span style="color:#B975FFFF">叛逆</span> = 1；<span style="color:#C6272EFF">唯我</span> = 0.5

    俘虏察觉判定：如果俘虏的毒术造诣 ≥ 为此物品解毒的毒术造诣需求，则俘虏察觉

    九品~一品毒药的解毒造诣需求 = 10/30/60/100/150/210/280/360/450

- 如果为触犯俘虏所属<span style="color:#6fb6ff">[门派戒律](../../../门派/门派概述/门派/门派-门派概述-门派.md)</span>的食物：俘虏好感-3000，抵抗意志增加10

- 如果不为以上两种情况：

  俘虏的抵抗意志减少 = 物品品级 × 俘虏立场影响

    俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 0.5；<span style="color:#9FE0DCFF">仁善</span> = 1；<span style="color:#FFFFFFFF">中庸</span> = 0；<span style="color:#B975FFFF">叛逆</span> = 1；<span style="color:#C6272EFF">唯我</span> = 0.5

  俘虏的好感增加 = 物品的赠礼好感 × 俘虏立场影响

    俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 0；<span style="color:#9FE0DCFF">仁善</span> = 0.1；<span style="color:#FFFFFFFF">中庸</span> = 0.05；<span style="color:#B975FFFF">叛逆</span> = 0.1；<span style="color:#C6272EFF">唯我</span> = 0

给予：

- 如果给予毒药：俘虏好感-3000，抵抗意志增加 = 物品品级 × 俘虏立场影响

  俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 0.5；<span style="color:#9FE0DCFF">仁善</span> = 1；<span style="color:#FFFFFFFF">中庸</span> = 0；<span style="color:#B975FFFF">叛逆</span> = 1；<span style="color:#C6272EFF">唯我</span> = 0.5

- 如果给予的不为毒药：俘虏的抵抗意志减少 = 物品品级/俘虏立场影响

  俘虏立场影响：<span style="color:#FFE78FFF">刚正</span> = 2；<span style="color:#9FE0DCFF">仁善</span> = 10；<span style="color:#FFFFFFFF">中庸</span> = 20；<span style="color:#B975FFFF">叛逆</span> = 10；<span style="color:#C6272EFF">唯我</span> = 2

劝说：

- 如果俘虏的抵抗意志 ≤ 0且身份品阶不高于侵袭进度的同道品阶限制，则俘虏加入太吾的同道；否则判定是否达到劝说成功条件。

- 如果劝说成功：如果俘虏的抵抗意志 ＞ 50，则抵抗意志-5；否则抵抗意志-10。

- 如果劝说失败：无事发生。

转移：

- 因互动影响的抵抗意志重置为与俘虏的<span style="color:#6fb6ff">[勇壮](../../../人物/属性/七元赋性/人物-属性-七元赋性.md)</span>相等。

处罚：

- 俘虏的好感-3000，戒心+200000。

- 如果太吾战胜，俘虏的抵抗意志+20；否则俘虏逃脱。

没收：

- 俘虏的抵抗意志增加 = 没收物品的心情变化值 × 2

- 俘虏的好感减少 = 没收物品的好感变化值 × 2

- 俘虏的心情减少 = 没收物品的心情变化值

询问：

- 根据俘虏的立场判定是否成功：

  <span style="color:#FFE78FFF">刚正</span>：需要抵抗意志 ≤ 10

  <span style="color:#9FE0DCFF">仁善</span>：需要抵抗意志 ≤ 30

  <span style="color:#FFFFFFFF">中庸</span>：需要抵抗意志 ≤ 40

  <span style="color:#B975FFFF">叛逆</span>：需要抵抗意志 ≤ 50

  <span style="color:#C6272EFF">唯我</span>：需要抵抗意志 ≤ 20

- 如果询问成功：随机获得俘虏持有的一条见闻或秘闻。若为见闻，俘虏抵抗意志+10；若为秘闻，俘虏抵抗意志+20。

- 如果询问失败：无事发生。
