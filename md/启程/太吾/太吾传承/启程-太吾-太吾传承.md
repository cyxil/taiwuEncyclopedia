# 太吾传承

> 百科 Key: `启程-太吾-太吾传承`
> 层级: 启程 / 太吾 / 太吾传承

在游戏过程中，太吾传人可能会迎来死亡或<span style="color:#6fb6ff">[入魔](../../相枢/相枢/启程-相枢-相枢.md#入魔)</span>，但这并不意味着游戏的结束。此时玩家需要将伏虞剑柄传给游戏中的其他人物，使其成为新一任的太吾传人。新任太吾将会继承前任太吾的所有功力和物品，随后玩家将扮演这名新任太吾来继续游戏。这一过程称为「太吾传承」。


太吾传承的条件：

- 太吾传承时，需要选择太吾的一名<span style="color:#6fb6ff">[同道](../../../交互/其他交互/同道/交互-其他交互-同道.md)</span>作为继任者。<span style="color:#6fb6ff">[婴儿](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#身龄)</span>不能作为继任者。

- 当同道中没有可以传承的人物时，若<span style="color:#6fb6ff">[世界细节](../../创建人物/世界细节/启程-创建人物-世界细节.md)</span>选项「随机继承人」已开启，则会随机传承给世界中的一名人物；若未开启，则游戏以失败告终。

- 如果尚未通过主线故事中的「古墓仙人」，无法进行太吾传承。


太吾传承的效果：

- 继任者的精纯、真气、技艺和武学的修习状况、志向状况、持有的物品和见闻均会被替换，变为与上一任太吾相同。

- 太吾传承时，可以使用上一任太吾的遗惠点数选择生平遗惠。

- 太吾传承时，可以调整世界细节的设置。每个世界细节只能增加或减少一个难度等级。无法变更「世界的人数」、「人物的寿元」的设置。


### 生平遗惠

太吾一生所经历的种种事迹会转化为生平遗惠点数。

太吾传承时，可以花费上一任太吾所累积的遗惠点数，启用生平遗惠，为继任的太吾提供各类加成。


获取遗惠点数：

- 太吾在游历世界时的各种行为，都可能累积相应的遗惠点数。

- 每一项遗惠点数达到获取上限后，直到下次太吾传承前，将不再继续获取此项遗惠点数。

- <span style="color:#6fb6ff">[世界细节](../../创建人物/世界细节/启程-创建人物-世界细节.md)</span>设置会影响相应类型的遗惠点数获取。世界细节难度越高，获取的遗惠点数越多。

获得生平遗惠：

每当一项遗惠点数累积至获取上限的50%和100%时，遗惠界面将会随机出现1个对应类型的生平遗惠，以供下次太吾传承时选择。

生平遗惠分为身难、修持、机缘三类。对应类型的世界细节难度越高，则出现的生平遗惠品级越高。

生平遗惠的效果分为永久和即时两种。

- 永久：此遗惠的效果将保留在<span style="color:#6fb6ff">[伏虞剑柄](../太吾传人/启程-太吾-太吾传人.md#伏虞剑柄)</span>中，为此后的每一代太吾传人都提供加成。

启用遗惠加成：

- 太吾传承时，可以从已获得的生平遗惠中，花费遗惠点数选择启用遗惠，为下一代太吾提供相应的加成。

- 太吾传承完成后，未启用的生平遗惠将会清空。

- 太吾传承完成后，未花费的遗惠点数将会以25%的比例转化为下一代太吾的威望。


获取随机遗惠：

- 进行太吾传承时，还可以消耗遗惠点数，获取额外的随机遗惠。以此方式获取的遗惠，无需再消耗遗惠点数即可启用其加成。

- 获取随机遗惠时，需要选择一种遗惠类型，并从该类型的随机三项遗惠中选择一项。

- 对应类型的世界细节难度越高，则能够获取的随机遗惠品级越高。

- 在一次太吾传承中，首次获取随机遗惠需要消耗500遗惠点数，后续每次获取所需的遗惠点数翻倍。
