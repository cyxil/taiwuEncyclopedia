# 相枢

> 百科 Key: `启程-相枢-相枢`
> 层级: 启程 / 相枢 / 相枢

相枢，意为「万相之核枢」，是太吾世界种种灾祸的根源，也是<span style="color:#6fb6ff">[太吾传人](../../太吾/太吾传人/启程-太吾-太吾传人.md)</span>的宿敌。


### 侵袭进度

侵袭进度代表相枢对世界的威胁程度。

侵袭进度越高，便有越多的人物可能被相枢夺取心神，但世人对太吾传人的重视程度也将越高。

当太吾通过挑战<span style="color:#6fb6ff">[剑冢](../剑冢/启程-相枢-剑冢.md)</span>提升<span style="color:#6fb6ff">[精纯境界](../../../修习/内力/精纯境界/修习-内力-精纯境界.md)</span>后，侵袭进度也将随之增加。

### 入魔

相枢会夺取太吾世界众生的心神，使其逐渐丧失理智，陷入癫狂。当人物的心神被相枢完全占据时，即称为「入魔」。

人物入魔后，将会丧失正常的行为能力，变为失心人袭击其他人物。

太吾传人同样有可能入魔。一旦太吾传人入魔，便会触发<span style="color:#6fb6ff">[太吾传承](../../太吾/太吾传承/启程-太吾-太吾传承.md)</span>。


当前侵袭进度决定了可能成为失心人的最高品阶。身份高于此品阶的人物不会入魔。

太吾传人入魔不受侵袭进度限制。


### 入魔程度

入魔程度表示人物心神受相枢侵蚀的程度。

随着入魔程度的累积，人物将逐渐变为「相枢入邪」状态，最终进入「相枢入魔」状态。

- 相枢入邪：当入魔程度达到100时，人物将获得「<span style="color:#9eb767">[相枢入邪](../../../词条/特性/相枢入邪.md)</span>」特性，<span style="color:#6fb6ff">[处世立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>暂时转变为<span style="color:#B975FFFF">叛逆</span>。

- 相枢入魔：若入魔程度达到200时，人物将获得「<span style="color:#9eb767">[相枢化魔](../../../词条/特性/相枢化魔.md)</span>」特性，处世立场暂时转变为<span style="color:#C6272EFF">唯我</span>，并脱离原本所属的势力，成为失心人。


增加入魔程度的方式：

- <span style="color:#6fb6ff">[心情](../../../人物/人物信息/心情/人物-人物信息-心情.md)</span>过低会导致入魔程度不断增加。

  月份更替时，若人物的心情处于<span style="color:#E9D382FF">沮丧</span>/<span style="color:#B975FFFF">痛苦</span>/<span style="color:#C6272EFF">悲极</span>，分别增加1/3/5点入魔程度。

- <span style="color:#6fb6ff">[袭击](../../../交互/人物互动/敌对/交互-人物互动-敌对.md#袭击)</span>其他人物时，将会增加5点入魔程度。

- 使用血犼教的门派恩义互动「<span style="color:#6fb6ff">[血池秘法](../../../交互/人物互动/门派恩义互动/交互-人物互动-门派恩义互动.md#血池秘法)</span>」、在战斗中使用<span style="color:#6fb6ff">[神剑碎片](../剑冢/启程-相枢-剑冢.md#神剑碎片)</span>，也会增加人物的入魔程度。

- 完成元山派地区故事后，可以通过「<span style="color:#6fb6ff">[三才护阵](../../../门派/门派一览/元山派/门派-门派一览-元山派.md#特有功能)</span>」增加人物的入魔程度。


减少入魔程度的方式：

- 使用伏虞剑柄救治失心人后，能够大幅减少其入魔程度。

- 通过元山派的门派恩义互动「<span style="color:#6fb6ff">[石牢静坐](../../../交互/人物互动/门派恩义互动/交互-人物互动-门派恩义互动.md#石牢静坐)</span>」，可以减少太吾或太吾同道的入魔程度。

- 通过道长志向技能「<span style="color:#6fb6ff">[驱邪法事](../../../游历/志向/道长/游历-志向-道长.md#驱邪法事)</span>」，驱邪成功后可以清除他人的入魔程度。

- 完成元山派地区故事后，可以通过「<span style="color:#6fb6ff">[三魔乱阵](../../../门派/门派一览/元山派/门派-门派一览-元山派.md#特有功能)</span>」减少人物的入魔程度。


### 失心人

已经入魔的人物，称为「失心人」。

失心人不具备正常的行为能力，并会在月份更替时袭击其他人物（如果在太吾附近，则优先袭击太吾）。

当太吾在战斗中击败失心人时，可以使用伏虞剑柄来救治失心人。

在战斗中使用伏虞剑柄后，太吾可以进行如下选择：

- <span style="color:#FFE78FFF">刚正</span>：以正言当头棒喝（救治失心人）

  对方的入魔程度变为0，心情变为0。

  太吾获得<span style="color:#6fb6ff">[伏虞心念](../../太吾/太吾传人/启程-太吾-太吾传人.md#伏虞心念)</span>和对方所属地区的<span style="color:#6fb6ff">[恩义](../../../世界/地图/州域与地区/世界-地图-州域与地区.md#地区恩义)</span>

- <span style="color:#9FE0DCFF">仁善</span>：助其收摄心神（救治失心人）

  对方的入魔程度变为99，心情变为0，好感+30000

  太吾获得<span style="color:#6fb6ff">[伏虞心念](../../太吾/太吾传人/启程-太吾-太吾传人.md#伏虞心念)</span>和对方所属地区的<span style="color:#6fb6ff">[恩义](../../../世界/地图/州域与地区/世界-地图-州域与地区.md#地区恩义)</span>

- <span style="color:#FFFFFFFF">中庸</span>：待其自行醒悟（救治失心人）

  对方的入魔程度变为50，心情变为0，好感+15000

  太吾获得<span style="color:#6fb6ff">[伏虞心念](../../太吾/太吾传人/启程-太吾-太吾传人.md#伏虞心念)</span>和对方所属地区的<span style="color:#6fb6ff">[恩义](../../../世界/地图/州域与地区/世界-地图-州域与地区.md#地区恩义)</span>

- <span style="color:#B975FFFF">叛逆</span>：扰乱其心神（放弃救治失心人）

  对方的心情变为0，好感变为0

- <span style="color:#C6272EFF">唯我</span>：为其了断（杀死失心人）

  对方死亡，太吾获得对方所属地区的<span style="color:#6fb6ff">[恩义](../../../世界/地图/州域与地区/世界-地图-州域与地区.md#地区恩义)</span>

如果选择救治失心人的选项，救治后可以邀请对方加入<span style="color:#6fb6ff">[同道](../../../交互/其他交互/同道/交互-其他交互-同道.md)</span>或任其自行离开。

任其自行离开后，对方将会成为附近城镇的乞丐。


失心人不会因为<span style="color:#6fb6ff">[健康](../../../人物/伤病/健康/人物-伤病-健康.md)</span>耗尽死亡。月份更替时，失心人的健康会恢复至上限。

月份更替时，世界各地的失心人可能被他人捕获，并押送至太吾村的<span style="color:#6fb6ff">[石屋](../../../产业/特殊建筑/太吾村/产业-特殊建筑-太吾村.md#石屋)</span>或所在州域的<span style="color:#6fb6ff">[监牢](../../../世界/势力/监牢/世界-势力-监牢.md)</span>。太吾也可以在石屋中处置关押的失心人。

月份更替时，世界中的随机1~3个失心人将会被捕获。

- 只有已开通<span style="color:#6fb6ff">[驿站](../../../世界/地图/州域与地区/世界-地图-州域与地区.md#旅行)</span>地区的失心人可能被捕获。

- 只有精纯低于太吾至少6点的失心人可能被捕获。

- 被捕获的失心人有75%的概率被押送至太吾村的石屋，有25%的概率被押送至所在州域的监牢。
