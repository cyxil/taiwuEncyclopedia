# 玄石火灰

> 百科 Key: `启程-相枢-玄石火灰`
> 层级: 启程 / 相枢 / 玄石火灰

玄石火灰，简称「玄灰」，是由相枢所造成的一种不治之症。

当一个<span style="color:#6fb6ff">[势力](../../../世界/势力/势力/世界-势力-势力.md)</span>的人物数量过多时，其中的人物便有可能沾染玄灰。

<span style="color:#6fb6ff">[太吾村](../../太吾/太吾传人/启程-太吾-太吾传人.md#太吾村)</span>不会受到玄灰侵蚀。


### 玄灰状况

玄灰状况表示玄灰对各个势力潜在的威胁程度，由轻到重分为：隐而未发、濒临发作、正在肆虐。


势力内的<span style="color:#6fb6ff">[成年](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#身龄)</span>人物数量相较于此势力的人口上限越多，则玄灰状况越严重。

- 不同势力的人口上限各有差异。

- <span style="color:#6fb6ff">[世界细节](../../创建人物/世界细节/启程-创建人物-世界细节.md)</span>「世界的人数」设置会影响各个势力的人口上限。


达到各级玄灰状况的条件：

- 隐而未发：成年人物数量不高于人口上限的75%

- 濒临发作：成年人物数量为人口上限的（75%，100%]

- 正在肆虐：成年人物数量超过人口上限

### 玄灰侵蚀

当势力的玄灰状况为「正在肆虐」时，便可能出现沾染玄灰的人物。

人物沾染玄灰后，将不断受到玄灰的侵蚀，并在一定时间后死亡。


人物的<span style="color:#6fb6ff">[命龄](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#命龄)</span>越高，沾染玄灰的概率越高。

以下人物不会沾染玄灰：

- <span style="color:#6fb6ff">[太吾传人](../../太吾/太吾传人/启程-太吾-太吾传人.md)</span>、<span style="color:#6fb6ff">[太吾村民](../../../产业/太吾村民/村民/产业-太吾村民-村民.md)</span>

- <span style="color:#6fb6ff">[未成年](../../../人物/人物信息/年龄/人物-人物信息-年龄.md#身龄)</span>的人物、处于<span style="color:#6fb6ff">[怀孕](../../../交互/其他交互/生育/交互-其他交互-生育.md#怀孕)</span>期间的人物

- <span style="color:#6fb6ff">[失心人](../相枢/启程-相枢-相枢.md#失心人)</span>、因奇书<span style="color:#6fb6ff">[执迷化魔](../../../修习/奇书宝典/奇书执迷/修习-奇书宝典-奇书执迷.md#执迷化魔)</span>的人物

- 服食了<span style="color:#9eb767">[活死药</span>](../../../词条/药毒/活死药.md)，具有<span style="color:#9eb767">[活死人](../../../词条/特性/活死人.md)</span>特性的人物

- <span style="color:#6fb6ff">[元山派](../../../门派/门派一览/元山派/门派-门派一览-元山派.md)</span>的地区故事中，受到<span style="color:#9eb767">[三才护命](../../../词条/特性/三才护命.md)</span>的人物

如果以上人物已经沾染了玄灰，则玄灰对此人物的侵蚀将会暂停。


沾染玄灰的人物将获得特性「<span style="color:#9eb767">[玄灰绝命](../../../词条/特性/玄灰绝命.md)</span>」，显示其剩余能够存活的时间。

- <span style="color:#6fb6ff">[世界细节](../../创建人物/世界细节/启程-创建人物-世界细节.md)</span>「人物的寿元」设置会影响沾染玄灰人物的剩余存活时间。

- 人物的<span style="color:#6fb6ff">[精纯](../../../修习/内力/精纯境界/修习-内力-精纯境界.md)</span>能够帮助其抵御玄灰侵蚀。精纯越高，剩余存活时间越久。


延缓玄灰侵蚀的方式：

- 太吾可以与沾染玄灰的人物进行「玄石火灰」互动，消耗<span style="color:#6fb6ff">[伏虞心念](../../太吾/太吾传人/启程-太吾-太吾传人.md#伏虞心念)</span>，为人物延缓玄灰侵蚀。

  每消耗1点伏虞心念，可以延缓1个月。

  每名人物最多可承受72点伏虞心念。

  为人物延缓玄灰后，将增加对方的好感，并获得对方的赠礼和地区恩义。

- 通过空桑派的特有功能「<span style="color:#6fb6ff">[鼎蛟淬身](../../../门派/门派一览/空桑派/门派-门派一览-空桑派.md#特有功能)</span>」，可使人物在10年内免受玄灰侵染。
