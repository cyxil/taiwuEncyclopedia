# 基本信息

> 百科 Key: `启程-创建人物-基本信息`
> 层级: 启程 / 创建人物 / 基本信息

开启新的旅程之前，玩家需要先创建一名人物，作为「<span style="color:#6fb6ff">[太吾传人](../../太吾/太吾传人/启程-太吾-太吾传人.md)</span>」，也就是游戏的主角。

玩家需要为新创建的人物设置<span style="color:#6fb6ff">[姓名](../../../人物/人物信息/姓名/人物-人物信息-姓名.md)</span>、<span style="color:#6fb6ff">[性别](../../../人物/人物信息/性别/人物-人物信息-性别.md)</span>、<span style="color:#6fb6ff">[年龄](../../../人物/人物信息/年龄/人物-人物信息-年龄.md)</span>、<span style="color:#6fb6ff">[生时](../../../人物/人物信息/生时/人物-人物信息-生时.md)</span>、<span style="color:#6fb6ff">[相貌](../../../人物/人物信息/魅力/人物-人物信息-魅力.md#相貌)</span>、<span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>等基本信息。

玩家还可以选择启用「自定义初始属性」，消耗属性点数，自行设置人物的初始<span style="color:#6fb6ff">[内力属性](../../../修习/内力/内力/修习-内力-内力.md#内力属性)</span>、<span style="color:#6fb6ff">[主要属性](../../../人物/属性/主要属性/人物-属性-主要属性.md)</span>、<span style="color:#6fb6ff">[特性](../../../人物/属性/特性/人物-属性-特性.md)</span>、技艺和武学<span style="color:#6fb6ff">[资质](../../../人物/人物信息/资质/人物-人物信息-资质.md)</span>，而不再随机生成。


创建人物时，需要设置人物的以下基本信息：

- <span style="color:#6fb6ff">[姓名](../../../人物/人物信息/姓名/人物-人物信息-姓名.md)</span>：人物的姓氏和名字。人物<span style="color:#6fb6ff">[生育](../../../交互/其他交互/生育/交互-其他交互-生育.md)</span>的后代可以继承人物的姓氏。

- <span style="color:#6fb6ff">[性别](../../../人物/人物信息/性别/人物-人物信息-性别.md)</span>：人物的性别，会影响人物结成爱慕或夫妻<span style="color:#6fb6ff">[关系](../../../交互/交互信息/关系/交互-交互信息-关系.md)</span>。

  性别还会影响人物在某些门派的请教和修习。

  少林派不会向女子传授武学；璇女派不会向男子传授武学。

  金刚宗以男子为尊，男子研读金刚宗秘籍更为便利；璇女派以女子为尊，女子研读璇女派秘籍更为便利。

- <span style="color:#6fb6ff">[年龄](../../../人物/人物信息/年龄/人物-人物信息-年龄.md)</span>：人物的年龄。新创建的人物年龄只能设置为16~30岁。

- <span style="color:#6fb6ff">[生时](../../../人物/人物信息/生时/人物-人物信息-生时.md)</span>：人物的出生月份，决定人物初始的<span style="color:#6fb6ff">[内力属性](../../../修习/内力/内力/修习-内力-内力.md#内力属性)</span>。

  不同门派的功法，会有不同的五行倾向，选择契合出生地门派功法的内力属性，将能更顺利地运用其功法。

- <span style="color:#6fb6ff">[相貌](../../../人物/人物信息/魅力/人物-人物信息-魅力.md#相貌)</span>：相貌美丑决定人物的<span style="color:#6fb6ff">[魅力](../../../人物/人物信息/魅力/人物-人物信息-魅力.md)</span>高低，从而影响他人的<span style="color:#6fb6ff">[好感](../../../人物/人物信息/好感/人物-人物信息-好感.md)</span>。

- 体型、肤色：仅改变人物的外观，没有其他实际影响。

- <span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>：人物的处世态度和观念，立场相近的人物更容易相处。

  不同门派会有不同的立场倾向，选择与出生地门派相同的立场，将能更顺利地与此门派的人物交往。

如果是已经游玩过《太吾绘卷》的玩家，可以用此前旅程中<span style="color:#6fb6ff">[铭刻](../../太吾/太吾传人/启程-太吾-太吾传人.md#铭刻)</span>下来的人物进行创建，或令铭刻人物作为芸芸众生的一员，参与到新的<span style="color:#6fb6ff">[世界演化](../../../世界/地图/月份更替/世界-地图-月份更替.md)</span>中。
