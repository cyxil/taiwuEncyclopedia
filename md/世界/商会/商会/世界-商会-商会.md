# 商会

> 百科 Key: `世界-商会-商会`
> 层级: 世界 / 商会 / 商会

商会是交易买卖各类商品的组织。

太吾世界中共有七大商会，这些商会各自售卖特定类型的货物：

- 服牛帮：售卖<span style="color:#6fb6ff">[食物](../../../物品/物品类型/食物/物品-物品类型-食物.md)</span>、食材<span style="color:#6fb6ff">[引子](../../../物品/物品类型/材料/物品-物品类型-材料.md#引子)</span>、<span style="color:#6fb6ff">[驮兽](../../../物品/物品类型/行装/物品-物品类型-行装.md#代步)</span>和各类杂物。

- 文山书海阁：售卖各类<span style="color:#6fb6ff">[技艺书籍](../../../物品/物品类型/书籍/物品-物品类型-书籍.md)</span>。

- 五湖商会：售卖<span style="color:#6fb6ff">[茶酒](../../../物品/物品类型/食物/物品-物品类型-食物.md)</span>、<span style="color:#6fb6ff">[口袋](../../../物品/物品类型/行装/物品-物品类型-行装.md#口袋)</span>、<span style="color:#6fb6ff">[绳索](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#绳索)</span>、各类装备<span style="color:#6fb6ff">[引子](../../../物品/物品类型/材料/物品-物品类型-材料.md#引子)</span>。

- 大武魁商号：售卖各类<span style="color:#6fb6ff">[武具](../../../物品/物品类型/武具/物品-物品类型-武具.md)</span>。

- 回春堂：售卖各类<span style="color:#6fb6ff">[丹药](../../../物品/物品类型/药毒/物品-物品类型-药毒.md#丹药)</span>、<span style="color:#6fb6ff">[毒药](../../../物品/物品类型/药毒/物品-物品类型-药毒.md#毒药)</span>。

- 公输坊：售卖<span style="color:#6fb6ff">[舟车](../../../物品/物品类型/行装/物品-物品类型-行装.md#代步)</span>、各类<span style="color:#6fb6ff">[建筑心材](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#心材)</span>。

- 奇货斋：售卖各类<span style="color:#6fb6ff">[武具](../../../物品/物品类型/武具/物品-物品类型-武具.md)</span>、<span style="color:#6fb6ff">[衣装](../../../物品/物品类型/行装/物品-物品类型-行装.md#衣装)</span>。

各大商会均有一个总部和一个分部，分别位于不同的<span style="color:#6fb6ff">[州府](../../势力/势力/世界-势力-势力.md)</span>。

在<span style="color:#6fb6ff">[铸剑山庄](../../../门派/门派一览/铸剑山庄/门派-门派一览-铸剑山庄.md)</span>的地区故事中，五湖商会将于西域-秦州建立新的分部「五湖珍宝」，可在其中购买<span style="color:#6fb6ff">[西域珍宝](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#西域珍宝)</span>。

### 交易

通过与商会、商队、商人进行交易，太吾可以购买商品，或出售自身持有的物品。

太吾可以通过以下几种形式进行交易：

- 商会建筑：即商会的总部和分部，这些建筑售卖的商品最为丰富。在州府的<span style="color:#6fb6ff">[产业视图](../../../产业/产业/产业视图/产业-产业-产业视图.md)</span>中，与商会建筑交互，即可进行交易。

- 商队：各大商会定期派遣商队，这些商队将在各个地区之间移动。在商队所在的地格与其交互，即可进行交易。

- 商人：各城镇势力均有身份为商人的人物。这些商人各自从属于某一个商会，与其进行「<span style="color:#6fb6ff">[浏览货物](../../../交互/人物互动/身份互动/交互-人物互动-身份互动.md#浏览货物)</span>」的互动，即可进行交易。

  太吾也可以在<span style="color:#6fb6ff">[传承名谱](../../../产业/特殊建筑/太吾氏祠堂/产业-特殊建筑-太吾氏祠堂.md)</span>中授予<span style="color:#6fb6ff">[商人](../../../产业/太吾村民/商人/产业-太吾村民-商人.md)</span>身份，并指定其从属的商会。


购回：太吾出售的物品可以在当月内，以出售时的价格购回。月份更替时，可购回的物品将被清空。

去旧呈新：与商会建筑、商人交易时，可以消耗5精力更新当前商店中的商品。


### 价格

交易时，商品的价格取决于其<span style="color:#6fb6ff">[价值](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#价值)</span>，同时受到其他因素的影响。

太吾购买的基础价格为物品价值的2倍。

太吾出售的基础价格为物品价值的20%（世界细节设置为普通时）。<span style="color:#6fb6ff">[世界细节](../../../启程/创建人物/世界细节/启程-创建人物-世界细节.md)</span>「世界的资源」设置会影响太吾出售物品的价格。


与商人交易时，商品的价格还可能受到对方的<span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>与<span style="color:#6fb6ff">[好感](../../../人物/人物信息/好感/人物-人物信息-好感.md)</span>的影响。

- 立场：太吾购买商品时，<span style="color:#9FE0DCFF">仁善</span>的商人更倾向于降价；<span style="color:#C6272EFF">唯我</span>的商人更倾向于涨价。

- 好感：商人的好感越高，太吾购买商品的价格越低，出售商品的价格越高。

### 额外商品

商店可能会出售一些额外商品，这些商品有特殊的标识。

- 每级商店会出售一件比当前<span style="color:#6fb6ff">[商店等级](../商店等级/世界-商会-商店等级.md)</span>高一级的商品，此商品的售价增加50%。

- 每级商店会出售一些当前季节对应的额外商品。「春日集会」奇遇的商店会出售随机一个季节的额外商品。

- 启用富商志向技能「<span style="color:#6fb6ff">[诸会宝号](../../../游历/志向/富商/游历-志向-富商.md#诸会宝号)</span>」后，每级商店会出售大量的额外商品。
