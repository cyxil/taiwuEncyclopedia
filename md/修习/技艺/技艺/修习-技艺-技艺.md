# 技艺

> 百科 Key: `修习-技艺-技艺`
> 层级: 修习 / 技艺 / 技艺

技艺是太吾世界中各种生活技能的统称。

技艺共分为16种：音律、弈棋、诗书、绘画、术数、品鉴、锻造、制木、医术、毒术、织锦、巧匠、道法、佛学、厨艺、杂学。

每种技艺类型均包括9门不同品级的技艺。

人物需要通过<span style="color:#6fb6ff">[研读](../../修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>技艺书籍以修习各种技艺，并解锁对应的建筑、技艺见闻和较艺策略。

- <span style="color:#6fb6ff">[建筑](../../../产业/产业/建筑/产业-产业-建筑.md)</span>：部分技艺书籍的第一篇研读完成后，便会解锁相应的建筑，解锁后即可在太吾村建造。

- <span style="color:#6fb6ff">[技艺见闻](../../../交互/交互信息/见闻/交互-交互信息-见闻.md#技艺见闻)</span>：每门技艺均对应1条技艺见闻，研读技艺书籍的相应篇章后即可获得。

  不同技艺见闻对应的篇章各不相同。

  技艺见闻无法重复获得。

- <span style="color:#6fb6ff">[较艺策略](../较艺/修习-技艺-较艺.md#较艺策略)</span>：当人物研读完成每种技艺类型的<span style="color:#6fb6ff">[下三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>、<span style="color:#6fb6ff">[中三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>、<span style="color:#6fb6ff">[上三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>中的任意一本技艺书籍后，将分别解锁1个对应的较艺策略。


人物每研读完技艺书籍的一个篇章后，便可以提高对应的技艺造诣。

技艺造诣主要影响以下方面：

- 研读：人物的技艺造诣影响其<span style="color:#6fb6ff">[研读](../../修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>相应技艺书籍的效率；最高一项技艺造诣决定在研读书籍时，可放置的<span style="color:#6fb6ff">[参考书籍](../../修习方式/研读书籍/修习-修习方式-研读书籍.md#参考书籍)</span>数量。

- 较艺：人物的技艺造诣决定其在较艺时的总<span style="color:#6fb6ff">[论据](../较艺/修习-技艺-较艺.md#论据)</span>。

- 战斗：人物的技艺造诣影响需要相应技艺的功法、兵器的<span style="color:#6fb6ff">[发挥威力](../../武学/功法/修习-武学-功法.md#威力)</span>。

- 诊疗：人物的医术、毒术造诣决定其<span style="color:#6fb6ff">[诊疗](../../../人物/伤病/诊疗/人物-伤病-诊疗.md)</span>的能力，以及每月能够进行疗伤、驱毒的次数。

- 产业：<span style="color:#6fb6ff">[太吾村民](../../../产业/太吾村民/村民/产业-太吾村民-村民.md)</span>的技艺造诣决定产业经营时的<span style="color:#6fb6ff">[工作效率](../../../产业/产业/产业经营/产业-产业-产业经营.md#工作效率)</span>，以及其履行<span style="color:#6fb6ff">[村民身份](../../../产业/太吾村民/村民/产业-太吾村民-村民.md#村民身份)</span>职能的效率。

- 制造：人物的技艺造诣决定其对相应类型物品进行<span style="color:#6fb6ff">[制造加工](../../../物品/制造加工/制造建筑/物品-制造加工-制造建筑.md)</span>的能力。

- 志向：人物的技艺造诣影响相应的<span style="color:#6fb6ff">[志向资历](../../../游历/志向/志向/游历-志向-志向.md#志向资历)</span>的获取效率。

- 交互：人物的技艺造诣影响其发起各种交互行为的概率和成功率。
