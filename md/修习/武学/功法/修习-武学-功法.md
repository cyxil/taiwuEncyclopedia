# 功法

> 百科 Key: `修习-武学-功法`
> 层级: 修习 / 武学 / 功法

武学是战斗中所用的各类功法的统称。

武学共分为14种：内功、身法、绝技、拳掌、指法、腿法、暗器、剑法、刀法、长兵、奇门、软兵、御射、乐器。

除内功、身法、绝技外，其余类型的功法又统称为摧破功法。

- 内功：内功是武学修习的根基，也是人物<span style="color:#6fb6ff">[内力](../../内力/内力/修习-内力-内力.md)</span>的主要来源。

  内功能够提供额外的<span style="color:#6fb6ff">[运功栏位](../运功/修习-武学-运功.md#运功栏位)</span>。

  内功能够进行<span style="color:#6fb6ff">[周天运转](../../修习方式/周天运转/修习-修习方式-周天运转.md)</span>，以获取内力和周天真气。

  内功能够提供被动生效的<span style="color:#6fb6ff">[心法效果](修习-武学-功法.md#心法效果)</span>、<span style="color:#6fb6ff">[运功效果](修习-武学-功法.md#运功效果)</span>，无须施展即可生效。

- 身法：<span style="color:#6fb6ff">[身法](../../../战斗/距离与移动/身法/战斗-距离与移动-身法.md)</span>又称轻灵功法，是在战斗中主动施展，帮助人物移动的功法。

- 绝技：绝技分为护体和奇窍。

  <span style="color:#6fb6ff">[护体功法](../../../战斗/攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>是在战斗中主动施展，以防御敌人的攻击的功法。

  奇窍功法能够提供被动生效的心法效果、运功效果，无须施展即可生效。

- 摧破：<span style="color:#6fb6ff">[摧破功法](../../../战斗/攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>是在战斗中主动施展，以攻击敌人的功法。

  每个门派都有各自擅长的摧破类型，不同的摧破类型代表不同的攻击方式和风格。


人物需要通过<span style="color:#6fb6ff">[研读](../../修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>功法书籍并进行<span style="color:#6fb6ff">[突破](../突破/修习-武学-突破.md)</span>，以修习各种功法。突破完成的功法可以提高对应的武学造诣，同类型且同品级的功法不会重复提高造诣。

武学造诣主要影响以下方面：

- 研读：人物的武学造诣影响其<span style="color:#6fb6ff">[研读](../../修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>相应功法书籍的效率。

- 周天：人物的最高一项武学造诣决定在周天运转时，可放置的<span style="color:#6fb6ff">[辅助内功](../../修习方式/周天运转/修习-修习方式-周天运转.md#辅助内功)</span>数量。

- 精纯：除太吾以外的人物，最高一项武学造诣决定其能够达到的<span style="color:#6fb6ff">[精纯境界](../../内力/精纯境界/修习-内力-精纯境界.md)</span>。

- 战斗：人物的武学造诣影响需要相应武学的功法、兵器的<span style="color:#6fb6ff">[发挥威力](修习-武学-功法.md#威力)</span>。

### 五行属性

每门功法均有各自的五行属性。

功法的五行属性共分为6种：<span style="color:#BA9140FF">金刚</span>（金）、<span style="color:#4F82B7FF">玄阴</span>（水）、<span style="color:#8757B3FF">紫霞</span>（木）、<span style="color:#C94B4BFF">纯阳</span>（火）、<span style="color:#81995EFF">归元</span>（土）、<span style="color:#858585FF">混元</span>。

不同五行属性的功法，其威力会受到人物的内力属性影响。详见「<span style="color:#6fb6ff">[内力属性](../../内力/内力/修习-内力-内力.md#内力属性)</span>」。

若功法的五行属性与人物的内力属性冲克，在战斗中主动施展此功法时，会受到<span style="color:#6fb6ff">[反噬](../../../战斗/攻击与防御/伤害/战斗-攻击与防御-伤害.md#反噬)</span>伤害和<span style="color:#6fb6ff">[内息紊乱](../../../人物/伤病/内息/人物-伤病-内息.md#内息紊乱)</span>。


### 运功效果

运功效果是将此功法放置于<span style="color:#6fb6ff">[运功栏位](../运功/修习-武学-运功.md#运功栏位)</span>中，即可生效的效果，能够增加人物的各种属性。

运功效果的来源有以下几种：

- <span style="color:#6fb6ff">[心法](../突破/修习-武学-突破.md#心法)</span>：内功、身法、奇窍功法的部分篇章心法可以提供运功效果。此效果在研读完对应篇章并领悟心法后即可生效。

- <span style="color:#6fb6ff">[玄机](../玄机/修习-武学-玄机.md)</span>：突破完成后，在玄机格中放置的玄机之物可以提供运功效果。

- <span style="color:#6fb6ff">[真气](../../内力/真气/修习-内力-真气.md)</span>：人物的真气会为内功提供额外的运功效果。真气总量越高，效果越强。


### 施展效果

施展效果是需要在战斗中主动施展此功法，才会生效的效果。

需要主动施展的功法类型有：<span style="color:#6fb6ff">[身法](../../../战斗/距离与移动/身法/战斗-距离与移动-身法.md)</span>、<span style="color:#6fb6ff">[摧破功法](../../../战斗/攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>、<span style="color:#6fb6ff">[护体功法](../../../战斗/攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>。

各类型功法的施展效果，详见对应的条目。


### 心法效果

心法效果是已突破的功法具有的特殊效果，分为正练和逆练。

心法效果的正逆练取决于所参悟的<span style="color:#6fb6ff">[心法](../突破/修习-武学-突破.md#心法)</span>。如果已领悟的正练心法数量大于逆练心法，则获得正练效果；反之获得逆练效果。

- 人物可以随时更换所参悟的心法，以改变心法效果的正逆练。

- 只有放置于<span style="color:#6fb6ff">[运功栏位](../运功/修习-武学-运功.md#运功栏位)</span>的功法，其心法效果才能生效。


心法效果乘区描述规则：

当功法效果影响某项数值（如属性、伤害等）时，可能分为以下三种计算方式，在效果描述中以不同的关键词体现：

- 加算：通常描述为「增加/减少」，计算方式为：基础值 + 加算值。

- 乘算：通常描述为「提高/降低」，计算方式为：基础值 × （1 + 乘算值%）。

- 最终结算：通常描述为「最终……提高/降低」。当有多个最终结算效果时，仅取其中加成的最大值与减成的最小值结算。计算方式为：基础值 × （1 + 最大加成值% + 最小减成值%）。

当多个乘区共同计算时，计算方式为：实际值 = （基础值 + 加算值之和）× （1 + 乘算值之和%） × （1 + 最大加成值% + 最小减成值%）。


### 施展需要

施展需要是在战斗中施展此功法时所需的前提条件，分为施展消耗和施展部位。

施展消耗：若不能满足施展消耗，则无法施展此功法。

- 身法：消耗<span style="color:#6fb6ff">[脚力](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#脚力)</span>。

- 摧破功法：消耗<span style="color:#6fb6ff">[提气](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#提气)</span>、<span style="color:#6fb6ff">[架势](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#架势)</span>、<span style="color:#6fb6ff">[蓄式](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#蓄式)</span>。

  腿法不消耗蓄式，但需要消耗<span style="color:#6fb6ff">[脚力](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#脚力)</span>。

  暗器、奇门（符咒）、御射类功法还需要消耗<span style="color:#6fb6ff">[兵器](../../../物品/物品类型/武具/物品-物品类型-武具.md#兵器)</span>的耐久。

- 护体功法：消耗<span style="color:#6fb6ff">[提气](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#提气)</span>、<span style="color:#6fb6ff">[架势](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#架势)</span>。

施展部位：轻灵功法、摧破功法需要依赖特定的身体部位来施展。

- 若施展部位处于<span style="color:#6fb6ff">[重伤](../../../战斗/标记与状态/部位状态/战斗-标记与状态-部位状态.md#重伤)</span>状态，则此功法的<span style="color:#6fb6ff">[施展时间](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#施展时间)</span>翻倍。

- 若施展部位的封穴达到3层，或处于<span style="color:#6fb6ff">[残毁](../../../战斗/标记与状态/部位状态/战斗-标记与状态-部位状态.md#残毁)</span>状态，则此功法无法施展。


### 发挥需求

发挥需求是发挥此功法的威力所需要的<span style="color:#6fb6ff">[主要属性](../../../人物/属性/主要属性/人物-属性-主要属性.md)</span>、技艺或武学<span style="color:#6fb6ff">[造诣](../../../人物/人物信息/造诣/人物-人物信息-造诣.md)</span>。

人物的当前主要属性和造诣超出发挥需求越多，则功法的发挥威力越高。


实战发挥是一种通用的发挥需求，表示此功法经历实战训练而获得的熟练度。

低于实战发挥的一项其他发挥，会被实战发挥取代，影响功法的发挥威力。

实战发挥的提升方式：

- 每当战斗结束后，在本场战斗中施展过或被动生效的所有功法的实战发挥均会提升。

- 通过在练功房中<span style="color:#6fb6ff">[练习功法](../../../产业/特殊建筑/练功房/产业-特殊建筑-练功房.md#练习功法)</span>，可以提升指定功法的实战发挥。


### 威力

功法威力决定功法的运功效果和施展效果的强弱。

功法的发挥威力表示由此功法的发挥需求提供的威力。发挥威力不能超过此功法的威力上限。

功法的威力上限初始为100%，通过<span style="color:#6fb6ff">[突破](../突破/修习-武学-突破.md)</span>可以增加威力上限。

功法威力影响以下效果的强弱：

- 所有功法的运功效果均受功法威力影响。

- 摧破：施展效果中的攻击属性、命中属性、施加的毒素量。

- 身法：施展效果中的移动速度、命中属性。

- 护体：施展效果中的防御属性、化解属性、反击威力、反震威力。


### 内外功比例

内外功比例决定施展此功法需要消耗的<span style="color:#6fb6ff">[提气](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#提气)</span>、<span style="color:#6fb6ff">[架势](../../../战斗/战斗基础/施展功法/战斗-战斗基础-施展功法.md#架势)</span>比例。如果为摧破功法，还会决定此功法的<span style="color:#6fb6ff">[破气](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#攻击属性)</span>、<span style="color:#6fb6ff">[破体](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#攻击属性)</span>比例，以及造成的内伤、外伤的比例。

- 功法的内功比例越高，施展时越倾向于消耗提气，造成的伤势越倾向于内伤。

- 功法的外功比例越高，施展时越倾向于消耗架势，造成的伤势越倾向于外伤。

人物可以随时调节功法的内外功比例。人物的<span style="color:#6fb6ff">[内功发挥](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#次要属性)</span>越高，可调节的范围越广。


内外功比例可调节范围计算公式：

可调节范围 = 基础可调节范围 × 内功发挥%
