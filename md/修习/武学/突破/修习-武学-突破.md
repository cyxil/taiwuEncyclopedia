# 突破

> 百科 Key: `修习-武学-突破`
> 层级: 修习 / 武学 / 突破

突破表示对一门功法的完全掌握。

突破能够大幅提升功法的<span style="color:#6fb6ff">[威力上限](../功法/修习-武学-功法.md#威力)</span>，并赋予<span style="color:#6fb6ff">[心法效果](../功法/修习-武学-功法.md#心法效果)</span>。


### 突破流程

突破开始前，需要选择<span style="color:#6fb6ff">[总纲](修习-武学-突破.md#总纲)</span>和每个篇章的<span style="color:#6fb6ff">[心法](修习-武学-突破.md#心法)</span>各一种，若已研读的书籍篇章数量不足，则无法开始突破。

突破时，需要从任意起点格出发，依次打通相邻的突破格，直至打通终点格，即可完成突破。

每个打通的突破格可以在突破完成后，增加此功法的<span style="color:#6fb6ff">[威力上限](../功法/修习-武学-功法.md#威力)</span>。打通的<span style="color:#6fb6ff">[玄机](../玄机/修习-武学-玄机.md)</span>格可以放置玄机之物，为此功法提供额外的威力上限和<span style="color:#6fb6ff">[参悟玄机](../玄机/修习-武学-玄机.md#参悟玄机)</span>效果。

突破完成后，也可以随时更换已参悟的心法，以改变心法效果的正逆。


突破相关的其他功能：

- 回顾：可以查看上一次突破的详情，或更换玄机之物。

- 重修：可以清除突破进度，重新突破。每门功法每月限重修一次。

- 预设：用于储存不同的突破结果，可在不同突破结果之间随时切换。


突破功法对人物的相应武学资质有一定要求。功法的品级越高，突破资质要求越高。

如果人物的资质未达到突破的资质要求，<span style="color:#6fb6ff">[天资上限](修习-武学-突破.md#天资上限)</span>将会减少。

部分门派的<span style="color:#6fb6ff">[修习效果](../../../门派/门派概述/门派/门派-门派概述-门派.md#门派修习)</span>允许人物在突破此门派的功法时，用特定的技艺资质取代突破所需的武学资质。

### 总纲

每本功法书籍由1篇总纲和5篇心法组成。

总纲是功法书籍的目录纲要，分为五种：承、合、解、异、独。

突破时，根据所选择的总纲，将会对此次突破过程造成不同的影响。

突破完成后将无法再更换总纲，必须重修后才能重新选择总纲。


通过<span style="color:#6fb6ff">[请教](../../../交互/人物互动/修习/交互-人物互动-修习.md#私下请教)</span>获得功法书时，根据传授者的立场不同，功法书的总纲类型不同。

### 心法

功法书的每个篇章研读完成后，即可参悟对应的心法，获得此心法提供的效果。

只有参悟了总纲和每个篇章的心法各一种后，才能进行突破。


每个篇章的心法分为正练、逆练两种。从第一篇到第五篇分别为：

- 正练心法：修、思、源、参、藏。

- 逆练心法：用、奇、巧、化、绝。

已参悟的每篇心法均会提供参悟效果，参悟效果不需要突破即可生效。

突破完成后，根据已参悟的心法，获得功法的<span style="color:#6fb6ff">[心法效果](../功法/修习-武学-功法.md#心法效果)</span>。若正练心法的数量大于逆练心法，则获得正练效果；反之获得逆练效果。

### 突破格

突破界面由各类突破格组成。突破时，需要从任意起点格出发，依次打通相邻的突破格，直至打通终点格，即可完成突破。

每个突破格可能会提供一定的功法<span style="color:#6fb6ff">[威力上限](../功法/修习-武学-功法.md#威力)</span>。功法突破完成后，将获得所有已打通的突破格的威力上限。

突破格分为以下几类：

- 如常格：没有特殊效果的突破格。

- 特殊格：打通后会产生特殊效果的突破格。

- <span style="color:#6fb6ff">[玄机格](../玄机/修习-武学-玄机.md)</span>：打通玄机格后，可以在其中放置玄机之物，以获得对应的参悟玄机效果。

### 连接突破格

功法的品级越高，突破格的数量越多，从起点到终点需要的连接次数也就越多。

每次连接必须从上一个打通的突破格开始，选择与之相邻的突破格连接。如果连接成功，即打通此突破格。连接失败的突破格不能再次连接。

如果没有可连接的突破格，突破将自动失败。


每次连接突破格都需要消耗一定数量的历练（无论是否成功打通）。若历练不足，则无法继续突破。

突破的功法品级越高，每次连接消耗的历练越多。

如果当前连接次数超过天资上限，则每超出1次，消耗的历练就会增加10%。

**突破历练消耗**

| 功法品级 | 每格消耗历练 |
| --- | --- | 
| 一品 | 1300 |
| 二品 | 900 |
| 三品 | 640 |
| 四品 | 420 |
| 五品 | 270 |
| 六品 | 150 |
| 七品 | 80 |
| 八品 | 30 |
| 九品 | 10 |


连接成功率的影响因素：

- 功法的品级越高，则连接成功率越低。

- 人物的悟性越高，则连接成功率越高。

- 人物的<span style="color:#6fb6ff">[精纯境界](../../内力/精纯境界/修习-内力-精纯境界.md)</span>达到<span style="color:#9eb767">气完</span>后，可以使连接成功率增加25%。

- <span style="color:#6fb6ff">[门派支持度](../../../门派/门派概述/门派支持/门派-门派概述-门派支持.md)</span>达到50%时，可以在突破此门派功法时，使连接成功率增加10%。

- 部分<span style="color:#6fb6ff">[产业建筑](../../../产业/产业/建筑/产业-产业-建筑.md)</span>的效果可以额外提高连接成功率。

- <span style="color:#6fb6ff">[世界细节](../../../启程/创建人物/世界细节/启程-创建人物-世界细节.md)</span>设置「突破的难度」会额外增加或减少连接成功率。


连接成功率计算公式：

基础连接成功率% = （30 + 功法品级影响 + 悟性影响 + 门派支持影响） × 产业建筑影响 + 世界细节影响

- 功法品级影响 =  （9 - 功法品级） × 5

- 悟性影响 = 人物的悟性 ÷ 10（向下取整，最大30）

- 世界细节影响：详见「<span style="color:#6fb6ff">[世界细节](../../../启程/创建人物/世界细节/启程-创建人物-世界细节.md)</span>」条目的「突破的难度」

实际连接成功率：在基础连接成功率的基础上，进行上下15%的随机浮动，并计算特殊突破格的影响。


### 天资上限

天资上限为突破中可连接突破格的次数。当连接次数超过天资上限后，本次突破将进入走火入魔的状态。

如果人物的资质未达到突破的资质要求，天资上限将会减少。

通过提高<span style="color:#6fb6ff">[门派支持度](../../../门派/门派概述/门派支持/门派-门派概述-门派支持.md)</span>、修建产业建筑「<span style="color:#9eb767">玄冰壁</span>」，可以增加天资上限。


天资上限计算公式：

- 若人物资质达到功法的资质要求，天资上限 = 20

- 若人物资质为功法资质要求的[50%，100%），天资上限 = 15 × 人物资质 ÷ 资质要求

- 若人物资质未达到功法资质要求的50%，天资上限 = 10 × 人物资质 ÷ 资质要求


### 走火入魔

当连接突破格的次数超过天资上限后，本次突破将进入走火入魔的状态。

走火入魔状态下，每次连接突破格都会受到<span style="color:#6fb6ff">[伤势](../../../人物/伤病/伤势/人物-伤病-伤势.md)</span>并增加<span style="color:#6fb6ff">[内息紊乱](../../../人物/伤病/内息/人物-伤病-内息.md#内息紊乱)</span>。

如果部位伤势达到上限，则会损害人物的<span style="color:#6fb6ff">[健康](../../../人物/伤病/健康/人物-伤病-健康.md)</span>。当人物的健康为0时，将无法继续突破。


入魔上限为走火入魔状态下最多可以连接突破格的次数。入魔上限耗尽后，本次突破将直接失败。

- 入魔上限的基础连接次数等于天资上限，并受到所选总纲的影响。

- 若在走火入魔时获得额外的天资上限，则本次突破暂时脱离走火入魔状态，直至连接次数再次超过天资上限后，重新进入走火入魔状态。此前走火入魔时的连接次数不会清除。


玄关弱点表示此功法在走火入魔状态下每次连接突破格时，所受伤势的类型、数量和受伤部位。

走火入魔效果公式：

- 每次连接受到的伤势：类型和受伤部位如玄关弱点所示（随机选择其中一个身体部位）

  受伤数量：下三品/中三品/上三品功法，分别为1/2/3层

- 每次连接增加的内息紊乱 = 内功品级 × 5 + 10
