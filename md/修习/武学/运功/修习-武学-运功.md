# 运功

> 百科 Key: `修习-武学-运功`
> 层级: 修习 / 武学 / 运功

运功指的是将已习得的功法放置于运功栏位，以启用这些功法的效果。

只有已运功的功法才能在战斗中施展。任何功法的<span style="color:#6fb6ff">[运功效果](../功法/修习-武学-功法.md#运功效果)</span>和<span style="color:#6fb6ff">[心法效果](../功法/修习-武学-功法.md#心法效果)</span>也只有在运功后才能生效。

只要是已习得的功法即可运功，不需要研读或突破完成。


运功相关的其他功能：

- 预设：用于储存不同的运功方案，可在不同运功方案之间快速切换。

- 自动运功：可以按照一定规则自动填充运功栏位。

- 修改战斗界面功法排序：可以调整已运功的功法在战斗界面的排列顺序。


### 运功栏位

运功时，不同类型的功法需要放置于不同的运功栏位中。

运功栏位分为以下类型：

- 内功栏位：用于放置<span style="color:#6fb6ff">[内功](../功法/修习-武学-功法.md)</span>，以提供更多的运功栏位。

- 摧破栏位：用于放置<span style="color:#6fb6ff">[摧破功法](../../../战斗/攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>。

- 轻灵栏位：用于放置<span style="color:#6fb6ff">[身法](../../../战斗/距离与移动/身法/战斗-距离与移动-身法.md)</span>。

- 护体栏位：用于放置<span style="color:#6fb6ff">[护体功法](../../../战斗/攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>。

- 奇窍栏位：用于放置<span style="color:#6fb6ff">[奇窍功法](../功法/修习-武学-功法.md)</span>。

- 万用栏位：可以转化为除内功栏位以外的任意运功栏位。


人物初始拥有的内功栏位为6格，摧破、轻灵、护体、奇窍栏位各1格。

在内功栏位放置内功后，可以增加除内功栏位外的其他运功栏位数量。

人物的每种运功栏位上限均为9格。因人物特性额外增加的运功栏位不计入此上限，如<span style="color:#6fb6ff">[志向](../../../游历/志向/志向/游历-志向-志向.md)</span>特性、<span style="color:#6fb6ff">[九世轮回](../../../人物/人物信息/轮回/人物-人物信息-轮回.md)</span>特性等。


品级越高的功法，占用的栏位数量越多。<span style="color:#6fb6ff">[下三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[中三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[上三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>的功法分别占用1/2/3格运功栏位。

若运功栏位数量不足，则无法放置对应的功法。

运功栏位数量减少后，如果无法满足已运功的功法需要占用的栏位，将会导致这些功法失效。

通过精解功法，或<span style="color:#6fb6ff">[奇书效果](../../奇书宝典/解读奇书/修习-奇书宝典-解读奇书.md#奇书效果)</span>「收置」，可以减少高品级功法占用的运功栏位。


### 精解

精解指的是将功法去繁就简，减少其占用的运功栏位，但会大幅提高功法的<span style="color:#6fb6ff">[发挥需求](../功法/修习-武学-功法.md#发挥需求)</span>。

精解的功法会减少1格占用的运功栏位。但任何功法至少需占用1格运功栏位，原本只占用1格运功栏位的功法无法精解。

若精解内功，此内功将不再提供原本的运功栏位，改为根据此内功的品级提供万用栏位。<span style="color:#6fb6ff">[中三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[上三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>的内功精解后，分别提供2格/3格万用栏位。

精解功法后，此功法的发挥需求将会大幅提高。中三品/上三品的功法精解后，发挥需求分别提高150%/100%。
