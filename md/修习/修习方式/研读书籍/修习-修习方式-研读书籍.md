# 研读书籍

> 百科 Key: `修习-修习方式-研读书籍`
> 层级: 修习 / 修习方式 / 研读书籍

将书籍放置于「研读书籍」栏位中，即可进行研读。

通过研读书籍，可以学习技艺和功法。当一本技艺书研读完成后，人物便完全掌握了此技艺；当一本功法书研读完成后，人物便可以<span style="color:#6fb6ff">[突破](../../武学/突破/修习-武学-突破.md)</span>此功法。

研读的方式有以下几种：

- 通常研读：人物在每次<span style="color:#6fb6ff">[月份更替](../../../世界/地图/月份更替/世界-地图-月份更替.md)</span>时都会自动进行一次研读。

- <span style="color:#6fb6ff">[专心致志](修习-修习方式-研读书籍.md#专心致志)</span>：消耗悟性和精力，主动进行研读。

- <span style="color:#6fb6ff">[实战领悟](修习-修习方式-研读书籍.md#实战领悟)</span>：战斗或较艺结束后，有概率触发「实战领悟」，立即进行一次研读。

- <span style="color:#6fb6ff">[拾取遗宝](../../../世界/地图互动/地格互动/世界-地图互动-地格互动.md#拾取遗宝)</span>：拾取地格上的研读奖励，可以进行一次研读。

- 恩义互动：通过然山派恩义互动「<span style="color:#6fb6ff">[王禅典籍](../../../交互/人物互动/门派恩义互动/交互-人物互动-门派恩义互动.md#王禅典籍)</span>」，可以快速进行多次研读。

通常每进行一次研读，研读的书籍和所有参考书籍均会损耗1点耐久。以实战领悟、拾取遗宝的方式研读书籍，不会损耗书籍的耐久。

启用才俊志向技能「<span style="color:#6fb6ff">[博古通今](../../../游历/志向/才俊/游历-志向-才俊.md#博古通今)</span>」，可以提高研读书籍的效率。


### 研读效率

研读效率取决于书籍的研读需求和各个篇章的完整程度。

人物的造诣超出研读需求越多，研读效率越高。

残缺、亡佚篇章的研读效率会大幅降低。已读完的书籍的研读效率会大幅提升。

- <span style="color:#6fb6ff">[参考书籍](修习-修习方式-研读书籍.md#参考书籍)</span>、<span style="color:#6fb6ff">[研读策略](修习-修习方式-研读书籍.md#研读策略)</span>可以提升研读效率。

- 部分<span style="color:#6fb6ff">[产业建筑](../../../产业/产业/建筑/产业-产业-建筑.md)</span>会为所在定居点影响范围内的所有人物提升研读效率。

- 才俊志向技能「<span style="color:#6fb6ff">[博古通今](../../../游历/志向/才俊/游历-志向-才俊.md#博古通今)</span>」可以提升研读效率。

- <span style="color:#6fb6ff">[世界细节](../../../启程/创建人物/世界细节/启程-创建人物-世界细节.md)</span>「研读的速度」设置会影响研读效率。


研读效率计算公式：

研读效率 = 篇章完整程度 × 未读篇章影响 × （1 + 造诣影响） × （1 + 参考书加成 + 研读策略加成 + 建筑加成 + 剑柄加成 + 志向加成） × （1 + 支持效果加成） × 世界细节影响

- 篇章完整程度：完整 = 50%；残缺 = 10%；亡佚 = 1%

- 未读篇章影响 = 50%^未读篇章数量

- 造诣影响 = 当前造诣 ÷ 研读需求，最小为10%，最大为1000%


### 研读进度

研读时会根据研读效率，按照书籍的篇章顺序依次增加研读进度。

当一个篇章的研读进度达到100%，则此篇章研读完成，继续增加下一篇章的研读进度。

当所有篇章的研读进度均达到100%后，此书籍研读完成。


每研读完一个篇章后，都会获得一定的<span style="color:#6fb6ff">[历练](../../../物品/物品与资源/资源/物品-物品与资源-资源.md#历练)</span>。对已研读完的书籍重复进行研读，仍会获得少量的历练。

研读获得历练：

九品~一品的书籍，每个篇章获得历练 =  20/100/200/300/450/600/800/1000/1300

重读已读完的书籍时，获得历练为原本的20%。


### 参考书籍

参考书籍可以提升研读效率、灵光一闪概率，并且能够补全篇章，使研读更加顺畅。

参考书籍需要放置于参考书籍栏位中。人物初始拥有一个参考书籍栏位，当人物的任意技艺造诣达到200、400时，将分别解锁第二、第三个栏位。

参考书籍的作用：

- 研读效率：每本参考书籍都会提升研读效率。

  参考书籍的品级越高，提升的研读效率越多。

  若参考书籍的技艺或武学类型与研读的书籍相同，将额外增加研读效率。

  当研读功法书籍时，放置此功法<span style="color:#6fb6ff">[发挥需求](../../武学/功法/修习-武学-功法.md#发挥需求)</span>中包含的技艺，且品级不低于此功法的技艺书籍作为参考书籍，将额外增加研读效率。

- 灵光一闪：若参考书籍为技艺书籍，将增加灵光一闪的触发概率。

  技艺书籍的品级越高，增加的概率越多。

- 研读策略：若参考书籍为技艺书籍，还会提供特定的研读策略。触发灵光一闪时，将必定出现这些研读策略。

  不同技艺类型的参考书籍，将提供不同的研读策略。

- 补全篇章：若参考书籍中有更为完整的篇章，则可以在研读时补全研读书籍中的相同篇章。

  仅当参考书籍与研读书籍为同一门技艺或功法，才能补全篇章。

  若为功法书籍，记载着相同<span style="color:#6fb6ff">[心法](../../武学/突破/修习-武学-突破.md#心法)</span>的篇章才会视为相同篇章。


### 灵光一闪

每次研读时，有一定概率触发灵光一闪。

在灵光一闪界面，可以使用研读策略，额外提升研读效率或研读进度。

- 触发后未使用的灵光一闪将会一直保留，直至更换研读的书籍。

- 若当前存在尚未使用的灵光一闪，则不会再次触发灵光一闪。

放置技艺书作为参考书籍，可以增加灵光一闪的触发概率。技艺书的品级越高，增加的概率越多。


### 研读策略

在灵光一闪界面，将随机出现9个可以使用的研读策略。

可以消耗悟性使用研读策略，来为书籍的各个篇章增加研读效率或研读进度。

书籍的每个篇章有3个研读策略栏位，使用后的研读策略将占用这些栏位，直到此策略的持续时间结束。研读策略栏位均被占用时，不能对此篇章继续使用研读策略。

### 专心致志

专心致志是主动研读书籍的方式，需要消耗悟性和精力。

通过消耗3悟性和1精力，可增长1点专心致志进度。专心致志的进度每达到10点，将进行一次研读。即使月份更替或更换研读的书籍，专心致志的进度也会保留。

每月最多可完成3次专心致志。本月已完成专心致志的次数越多，每次专心致志的研读效率越低。

专心致志的效率分为以下三种状态：

- 满：本月尚未完成专心致志，通过专心致志研读书籍的效率额外增加50%。

- 溢：本月已完成1次专心致志，通过专心致志研读书籍的效率不变。

- 疲：本月已完成2次专心致志，通过专心致志研读书籍的效率降低50%。

当处于疲状态下，再完成一次专心致志后，本月将不能继续专心致志。

月份更替后，专心致志将恢复为「满」状态。


通过专心致志研读时，不会触发灵光一闪。


### 实战领悟

战斗结束后，若研读的书籍为功法书籍，有一定概率触发实战领悟，立即进行一次研读。

较艺结束后，若研读的书籍为技艺书籍，有一定概率触发实战领悟，立即进行一次研读。


每月最多可通过战斗和较艺各触发一次实战领悟。

<span style="color:#6fb6ff">[战斗评价](../../../战斗/战斗基础/战斗流程/战斗-战斗基础-战斗流程.md#战斗评价)</span>、<span style="color:#6fb6ff">[较艺评价](../../技艺/较艺/修习-技艺-较艺.md#较艺评价)</span>越好，则触发实战领悟的概率越高。

以实战领悟的方式研读书籍，不会损耗书籍的耐久。
