# 精纯境界

> 百科 Key: `修习-内力-精纯境界`
> 层级: 修习 / 内力 / 精纯境界

精纯境界是通过累世修为而达到的武学境界，是决定人物战斗能力的重要因素。

精纯境界随着精纯的增长而提升。人物每获得2精纯，精纯境界便上升一层。

太吾需要通过攻破<span style="color:#6fb6ff">[剑冢](../../../启程/相枢/剑冢/启程-相枢-剑冢.md)</span>获得精纯，其他人物则通过提升自身的武学造诣获得精纯。


精纯的获取方式：

- 太吾拥有特性「<span style="color:#9eb767">[精纯残断](../../../词条/特性/精纯残断.md)</span>」，需要通过击败剑冢中的相枢化身获得精纯。

  每击败一名相枢化身，太吾便获得2精纯；若仅击败相枢化身的第一阶段，则获得1精纯。

  重复击败同一名化身的第一阶段，不会再次获得精纯。

  击败<span style="color:#6fb6ff">[破冢化身](../../../启程/相枢/剑冢/启程-相枢-剑冢.md#破冢化身)</span>不会获得精纯。

- 其他人物的精纯上限由其最高一项武学造诣决定。月份更替时，人物的精纯会逐渐增长至精纯上限。人物的身份品阶越高，精纯增长的速度越快。


精纯境界的作用：

- 精纯境界决定人物的<span style="color:#6fb6ff">[真气上限](../真气/修习-内力-真气.md#凝聚真气)</span>。

- 人物所达到的每个精纯境界均会提供各类加成效果。

- 太吾的精纯境界还决定太吾世界的<span style="color:#6fb6ff">[侵袭进度](../../../启程/相枢/相枢/启程-相枢-相枢.md#侵袭进度)</span>。

非太吾人物精纯上限计算公式：

从人物的所有武学精纯计数中，取最高一项作为精纯上限。

武学精纯计数 = 基础值 + （下三品加成 + 中三品加成 + 上三品加成） ÷ 10（向下取整）

- 基础值：人物此项武学的造诣界面中，每个品级的武学 + 1（九品除外）

- 下三品加成：人物此项武学的造诣界面中，选取下三品武学中最高的一项，九品：5；八品：10；七品：15

- 中三品加成：人物此项武学的造诣界面中，选取中三品武学中最高的一项，六品：20；五品：25；四品：30

- 上三品加成：人物此项武学的造诣界面中，选取上三品武学中最高的一项，三品：35；二品：45；一品：55


非太吾人物精纯提升速度计算公式：

人物精纯每月提升进度 = 40 + （人物身份品阶 + 1） × 45

提升每点精纯需要进度 = 精纯 × 400（从0精纯提升至1精纯需要400进度，从1精纯提升至2精纯需要800进度，依此类推）
