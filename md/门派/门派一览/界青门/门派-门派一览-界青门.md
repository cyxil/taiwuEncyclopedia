# 界青门

> 百科 Key: `门派-门派一览-界青门`
> 层级: 门派 / 门派一览 / 界青门

*界青门俗称人鬼同途之地，多收纳江湖恶徒。本门弟子行事隐秘，极少显露踪迹。门人最擅长奇门遁甲，飞针投石之术，倘若杀意一起，定教人死于无影无形。*


- 界青门位于淮南-界青崖地区，门派武学以<span style="color:#4F82B7FF">玄阴</span>属性为主。

- 界青门乃邪派外道，不会向<span style="color:#6fb6ff">[名誉](../../../人物/人物信息/名誉/人物-人物信息-名誉.md)</span>过高之人传授武学；界青门的敌对门派为武当派。

- 界青门<span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>以<span style="color:#B975FFFF">叛逆</span>为主，派中多<span style="color:#B975FFFF">叛逆</span>之士。


### 传授武学

*界青门武学**由指法、暗器、剑法组成，**往来悄然无踪，起势迅疾如电，发招凶狠利落，务以瞬息之间取人性命，绝无半分恻隐。*

界青门武学讲究另辟蹊径，界青门秘籍<span style="color:#6fb6ff">[逆练篇章](../../../修习/武学/突破/修习-武学-突破.md#心法)</span>的<span style="color:#6fb6ff">[研读](../../../修习/修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>效率提高50%；<span style="color:#6fb6ff">[正练篇章](../../../修习/武学/突破/修习-武学-突破.md#心法)</span>的研读效率降低50%。


界青门武学发挥所需的技艺：术数；弈棋（仅暗器功法需要）

太吾可以向界青门请教的技艺：弈棋、术数、毒术

### 地区故事

界青门的地区故事为《善恶无生》。

*界青门内，有一绝地，名曰无生渊。据传门人为接取委托，皆需亲下深渊，以鲜血拓印死令。凡接死令者为求成事不择手段，令闻者胆寒。*

*而今星象异变，门中暗流涌动，却不知这诡秘的界青门，究竟藏着何等不为人知的隐秘……*


触发条件：

- 已拔除四座<span style="color:#6fb6ff">[剑冢](../../../启程/相枢/剑冢/启程-相枢-剑冢.md)</span>

- <span style="color:#6fb6ff">[名誉](../../../人物/人物信息/名誉/人物-人物信息-名誉.md)</span>到达声驰千里或千夫所指

- 前往寿春城


完成后可获得：<span style="color:#9eb767">[星图残片</span>](../../../词条/宝物/星图残片.md)

后传故事开启条件：持有元山派宝物<span style="color:#9eb767">[绝魔骨</span>](../../../词条/宝物/绝魔骨.md)

完成后传故事可获得：<span style="color:#9eb767">[落霄遗曜</span>](../../../词条/宝物/落霄遗曜.md)


### 特有功能

完成界青门的地区故事《善恶无生》后，将解锁界青门的特有功能「奇纹星斗」。可杀死其他人物以夺取其星运，星运可用于在尚未进行<span style="color:#6fb6ff">[太吾传承](../../../启程/太吾/太吾传承/启程-太吾-太吾传承.md)</span>时获取<span style="color:#6fb6ff">[生平遗惠](../../../启程/太吾/太吾传承/启程-太吾-太吾传承.md#生平遗惠)</span>加成。

获取星运点数：

- 完成《善恶无生》后，太吾可以在支持度不低于30%的门派产业中建造建筑「奇纹星台」。每个门派仅能建造一座，总计最多建造5座。

- 每当建有「奇纹星台」的门派中的人物获得与其<span style="color:#6fb6ff">[门派正邪](../../门派概述/门派/门派-门派概述-门派.md)</span>不符的名誉时（正派人物获得负面名誉、邪派人物获得正面名誉、中立门派人物获得正面或负面名誉），此人物便会累积星运点数。人物所属势力变更后，其已累积的星运点数仍然保留。

- <span style="color:#6fb6ff">[精纯](../../../修习/内力/精纯境界/修习-内力-精纯境界.md)</span>越高的人物能够拥有的星运上限越多。已成年人物的星运点数达到上限的50%时，此人物将会成为「星运煞源」。

- 太吾可以通过与星运煞源进行「剥夺星运」互动，与其进行死斗。本场战斗中太吾会临时戴上面具，使对方无法察觉袭击者为太吾。太吾战胜并杀死星运煞源后，即可夺取此人物的所有星运点数。


奇纹星斗：

开启方式：使用行囊中的特殊物品「奇纹星斗」

使用效果：

- 在尚未进行太吾传承时，即可获取遗惠加成。以此法获取遗惠时，改为消耗星运点数，而非遗惠点数。

- 在奇纹星斗界面，太吾可以查看世界范围内的「星运煞源」位置。


完成《善恶无生》的后传故事，将使「奇纹星台」的建造总数上限增加至10座。
