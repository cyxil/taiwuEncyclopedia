# 血犼教

> 百科 Key: `门派-门派一览-血犼教`
> 层级: 门派 / 门派一览 / 血犼教

*相枢乱世以来，万恶之徒渐而聚首，创立了血犼教。教中魔功乱人本性，菩萨亦是难救，正派武林人士自然与其势不两立。*


- 血犼教位于江北-血犼谷地区，门派武学以<span style="color:#81995EFF">归元</span>属性为主。

- 血犼教乃邪派外道，不会向<span style="color:#6fb6ff">[名誉](../../../人物/人物信息/名誉/人物-人物信息-名誉.md)</span>过高之人传授武学；血犼教的敌对门派为百花谷。

- 血犼教<span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>以<span style="color:#C6272EFF">唯我</span>为主，派中多<span style="color:#C6272EFF">唯我</span>之士。


### 门派修习

*血犼教武学**由拳掌、指法、腿法、暗器组成，**邪诡恶毒，无所不用其极。甚或以人牲血祭，入于魔罗之道，而成不死邪功。*

血犼教武学邪诡恶毒，若修习者的<span style="color:#6fb6ff">[处世立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>为<span style="color:#B975FFFF">叛逆</span>或<span style="color:#C6272EFF">唯我</span>，血犼教秘籍的<span style="color:#6fb6ff">[研读](../../../修习/修习方式/研读书籍/修习-修习方式-研读书籍.md)</span>效率提高50%：若修习者的处世立场为<span style="color:#FFE78FFF">刚正</span>或<span style="color:#9FE0DCFF">仁善</span>，血犼教秘籍的研读效率降低50%。（此效果无法对<span style="color:#6fb6ff">[总纲](../../../修习/武学/突破/修习-武学-突破.md#总纲)</span>生效）


血犼教武学发挥所需的技艺：毒术、杂学

太吾可以向血犼教请教的技艺：品鉴、医术、毒术、佛学、杂学

### 地区故事

血犼教的<span style="color:#6fb6ff">[地区故事](../../门派概述/地区故事/门派-门派概述-地区故事.md)</span>为《血冢遗姝》。

*“寒狐啸青冢，鬼火烧白杨”。传闻血犼教阴风萧飒，竟有九冥恶鬼魂魄不散，踯躅徘徊，引得扬州城内流言蜚语，一时甚嚣尘上，令人闻风丧胆。*

*然而鬼神之事，本是渺茫荒惑，若非亲眼所见，岂知真相如何……*


触发条件：

- 前往扬州城


完成后可获得：<span style="color:#9eb767">[金犼铁券</span>](../../../词条/宝物/金犼铁券.md)

后传故事开启条件：持有然山派宝物<span style="color:#9eb767">[青琅归玄印</span>](../../../词条/宝物/青琅归玄印.md)

完成后传故事可获得：<span style="color:#9eb767">[凤纹玉璜</span>](../../../词条/宝物/凤纹玉璜.md)


### 特有功能

完成血犼教的地区故事《血冢遗姝》后，特殊人物「姬穸」将出现在地图上。如果姬穸好感为<span style="color:#F3802AFF">不渝</span>，可令其跟随太吾，消灭太吾附近的<span style="color:#6fb6ff">[外道](../../../世界/地图互动/敌人/世界-地图互动-敌人.md#外道)</span>与<span style="color:#6fb6ff">[任侠](../../../世界/地图互动/敌人/世界-地图互动-敌人.md#任侠)</span>，并为太吾恢复<span style="color:#6fb6ff">[健康](../../../人物/伤病/健康/人物-伤病-健康.md)</span>。

开启方式：与《血冢遗姝》特殊人物「姬穸」互动

姬穸饮血：

- 姬穸的身龄分为三个阶段：幼年、青年、成年。

- 姬穸每隔一段时间将杀死一名随机人物，并吸食其血。姬穸当前身龄阶段越年轻，则杀人饮血越频繁。每累积杀死7名人物，姬穸将会提升一个身龄阶段。

- 姬穸未跟随太吾时，将在世界范围内游荡，杀死随机地区的人物。姬穸跟随太吾时，饮血目标为太吾附近的人物。

- 姬穸不会杀死太吾的<span style="color:#6fb6ff">[亲友](../../../交互/交互信息/关系/交互-交互信息-关系.md#亲友)</span>或<span style="color:#6fb6ff">[太吾村民](../../../产业/太吾村民/村民/产业-太吾村民-村民.md)</span>，且优先杀死太吾的<span style="color:#6fb6ff">[仇敌](../../../交互/交互信息/关系/交互-交互信息-关系.md#仇敌)</span>。


姬穸随行：

- 如果姬穸好感为不渝，且当前太吾为初次与姬穸相遇的太吾或其转世，则可以令姬穸跟随太吾同行。

- 姬穸随行时，将自动消灭沿途地格的外道、任侠。

- 月份更替时，如果太吾的健康低于健康上限的33%，则姬穸将以自身变回幼年期为代价，为太吾恢复健康。青年期的姬穸可以恢复50%健康，成年期的姬穸可以恢复100%健康。


完成《血冢遗姝》的后传故事，姬穸杀人饮血的行为将变为吸食他人真气，并解锁「持印汲气」功能。姬穸可以协助太吾转移<span style="color:#6fb6ff">[内力五行](../../../修习/内力/内力/修习-内力-内力.md#内力五行)</span>，并与太吾相互传输<span style="color:#6fb6ff">[真气](../../../修习/内力/真气/修习-内力-真气.md)</span>。

开启方式：与《血冢遗姝》特殊人物「姬穸」互动

姬穸汲气：

- 太吾可在「持印汲气」界面，指定太吾本人或一名与太吾相识的成年人物，作为姬穸吸食真气的目标。若未指定人物，姬穸将选择其所在地格的随机人物为目标（不会选择太吾的亲友或太吾村民，且优先选择太吾的仇敌）。

- 姬穸每累积吸食14点真气后，将会提升一个身龄阶段。姬穸当前身龄阶段越年长，每月吸食的真气越多。

- 姬穸吸食的真气将会转化为姬穸自身的真气，每种真气最多50点。


姬穸互动：

- 在「持印汲气」界面，太吾可将姬穸的真气传输给自己作为<span style="color:#6fb6ff">[周天真气](../../../修习/修习方式/周天运转/修习-修习方式-周天运转.md#周天真气)</span>，也可将自身的周天真气传输给姬穸。

- 在「持印汲气」界面，太吾可指定一名与太吾相识的人物，令姬穸将其某种五行的内力转移给太吾，改变双方的内力五行比例。姬穸当前身龄阶段越年长，五行转移的效率越高。

- 在「持印汲气」界面，太吾可主动令姬穸为自己恢复健康，效果与姬穸自发行为相同。

- 当姬穸消灭外道时，将为太吾提供对应数量的地区恩义。
