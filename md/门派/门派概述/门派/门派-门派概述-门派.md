# 门派

> 百科 Key: `门派-门派概述-门派`
> 层级: 门派 / 门派概述 / 门派

门派是修习、传授<span style="color:#6fb6ff">[武学](../../../修习/武学/功法/修习-武学-功法.md)</span>的势力。

太吾世界中，共有十五大门派，分别为少林派、峨眉派、百花谷、武当派、元山派、狮相门、然山派、璇女派、铸剑山庄、空桑派、金刚宗、五仙教、界青门、伏龙坛、血犼教。

太吾传人因其身份所限，不能加入任何门派，但可以学习各大门派的功法和技艺。


门派有正邪之分，分为正道、中立、邪道。正道门派不会向<span style="color:#6fb6ff">[名誉](../../../人物/人物信息/名誉/人物-人物信息-名誉.md)</span>低于<span style="color:#8E8E8EFF">默默无闻</span>之人传授武学；邪道门派不会向名誉高于<span style="color:#8E8E8EFF">默默无闻</span>之人传授武学。

门派有<span style="color:#6fb6ff">[立场](../../../人物/人物信息/立场/人物-人物信息-立场.md)</span>之分，每个门派下的大多数人物都与本门派立场相同。

各个门派所擅长的武学不同，且每个门派的武学均以某一种<span style="color:#6fb6ff">[五行](../../../修习/武学/功法/修习-武学-功法.md#五行属性)</span>为主。


部分门派有特殊的戒律。这些门派的人物不会接受他人赠送的触犯戒律的物品。如果人物破戒，将会违反本门派的<span style="color:#6fb6ff">[法规](../../../世界/势力/法规/世界-势力-法规.md)</span>。

每个门派有各自的敌对门派。如果门派人物与敌对门派的人物结成正面关系，将会违反本门派的法规。门派人物违反本门法规后，为了躲避处罚，可能会逃往敌对门派。

### 学艺许可

学艺许可是向门派人物请教功法、技艺的前提。

当太吾来到门派所在的地格，送上拜帖后，即可取得此门派的学艺许可。如果在门派之外的地格上遇到此门派的<span style="color:#6fb6ff">[领袖](../../../世界/势力/势力/世界-势力-势力.md#身份)</span>人物，也可以取得学艺许可。

取得学艺许可后，太吾还需要取得门中各位弟子的支持，提升<span style="color:#6fb6ff">[门派支持度](../门派支持/门派-门派概述-门派支持.md)</span>，才能学习更高品级的功法和技艺。


### 门派互动

太吾可以与门派人物进行各种门派互动：

- <span style="color:#6fb6ff">[门派请教](../../../交互/人物互动/修习/交互-人物互动-修习.md#门派请教)</span>：学习该门派传授的功法和技艺。

- <span style="color:#6fb6ff">[交换门派藏书](../../../交互/人物互动/修习/交互-人物互动-修习.md#交换门派藏书)</span>：交换该门派传授的功法和技艺书。

- <span style="color:#6fb6ff">[获取支持](../../../交互/人物互动/亲近/交互-人物互动-亲近.md#获取支持)</span>：获取此人物的门派支持，不同立场的人物要求不同。

- <span style="color:#6fb6ff">[荐送弟子](../../../交互/人物互动/身份互动/交互-人物互动-身份互动.md#荐送弟子)</span>：引荐一名同道加入此门派。
