# 解封

> 百科 Key: `战斗-攻击与防御-解封`
> 层级: 战斗 / 攻击与防御 / 解封

解封是<span style="color:#6fb6ff">[铸剑山庄](../../../门派/门派一览/铸剑山庄/门派-门派一览-铸剑山庄.md)</span>功法特有的攻击方式。

释放解封攻击时，将对敌人造成不可避免的大量伤害，并根据兵器类型产生不同的效果。

**各类型兵器解封效果一览**

| 兵器类型 | 解封效果 |
| --- | --- | 
| 剑 | 攻击3次，每次将敌人的1个随机功法<span style="color:#6fb6ff">[封禁](../../标记与状态/封禁/战斗-标记与状态-封禁.md)</span>5秒 |
| 刀 | 攻击3次，对敌人造成的直接伤势变化为更难恢复的<span style="color:#6fb6ff">[旧伤](../../标记与状态/战败标记/战斗-标记与状态-战败标记.md#伤势标记)</span> |
| 长兵 | 攻击3次，<span style="color:#6fb6ff">[攻击范围](../攻击/战斗-攻击与防御-攻击.md#攻击范围)</span>变化为最大 |
| 掌套 | 攻击3次，每次吸取敌人20%的随机真气 |
| 暗器 | 攻击3次，分别对敌人造成1~3级的<span style="color:#6fb6ff">[破绽](../../标记与状态/部位状态/战斗-标记与状态-部位状态.md#破绽)</span> |
| 药霜、毒砂 | 攻击3次，施加的毒素为兵器的10倍 |
| 暗器、针匣、机关 | 攻击3次，分别对敌人造成1~3级的<span style="color:#6fb6ff">[封穴](../../标记与状态/部位状态/战斗-标记与状态-部位状态.md#封穴)</span> |
| 拂尘、长鞭 | 攻击3次，消除敌人当前运起的身法 |
| 对刺、短杵、令符 | 攻击3次，消除敌人当前运起的护体功法 |
| 箫笛、瑶琴 | 攻击3次，每次使敌人的<span style="color:#6fb6ff">[内息紊乱](../../../人物/伤病/内息/人物-伤病-内息.md#内息紊乱)</span>+30 |


铸剑山庄剑法、刀法、长兵功法的心法效果，会在战斗中为运用者装备的所有对应类型的兵器持续增加解封进度。

当兵器的解封进度达到100%时，即可释放此兵器的解封攻击。释放解封攻击后，兵器的解封进度重置为0%。

下三品/中三品/上三品的功法，每秒增加的解封进度分别为1.33%/2.00%/2.66%。同一类型的多个功法增加的进度不会叠加生效，只取其中的最大值。

除了解封剑、刀、长兵外，还可以通过铸剑山庄部分功法的心法效果，或<span style="color:#6fb6ff">[天枢玄铸](../../../门派/门派一览/铸剑山庄/门派-门派一览-铸剑山庄.md#特有功能)</span>机关人的助战指令「解封」，增加其他类型兵器的解封进度，以释放解封攻击。

- 解封攻击不会被敌人<span style="color:#6fb6ff">[化解](../攻击/战斗-攻击与防御-攻击.md#命中与化解)</span>，但也无法获得<span style="color:#6fb6ff">[架势](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md#架势)</span>和<span style="color:#6fb6ff">[蓄式](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md#蓄式)</span>。

- 以解封攻击造成<span style="color:#6fb6ff">[直接伤害](../伤害/战斗-攻击与防御-伤害.md#伤害来源)</span>时，必然<span style="color:#6fb6ff">[命中要害](../攻击/战斗-攻击与防御-攻击.md#命中要害)</span>。

- 当双方距离不在兵器的<span style="color:#6fb6ff">[攻击范围](../攻击/战斗-攻击与防御-攻击.md#攻击范围)</span>内时，无法释放解封攻击。

- <span style="color:#6fb6ff">[临机兵器](../../../物品/物品类型/武具/物品-物品类型-武具.md#兵器)</span>不会增加解封进度，也无法释放解封攻击。


解封攻击威力计算公式：

解封攻击威力 = 100% + 兵器威力 ÷5


### 生铸

生铸是<span style="color:#6fb6ff">[铸剑山庄](../../../门派/门派一览/铸剑山庄/门派-门派一览-铸剑山庄.md)</span>长兵功法特有的效果。

铸剑山庄长兵功法的心法效果，可以在运用者解封长兵时，将一件装备变为生铸神器，并赋予生铸特性。

如果生铸神器的品级高于原装备，则根据功法效果，需要额外消耗<span style="color:#6fb6ff">[精制材料](../../../物品/物品类型/材料/物品-物品类型-材料.md#精制材料)</span>才能生铸。


生铸神器的特点：

- 生铸神器拥有对应功法所赋予的生铸特性，效果见下表。

- 生铸神器的类型、材质与原装备相同，并且继承原装备的<span style="color:#6fb6ff">[装备特性](../../../物品/物品类型/装备/物品-物品类型-装备.md#装备特性)</span>、<span style="color:#6fb6ff">[精制](../../../物品/制造加工/精制/物品-制造加工-精制.md)</span>效果和<span style="color:#6fb6ff">[淬毒](../../../物品/制造加工/淬毒/物品-制造加工-淬毒.md)</span>状况。

- 生铸神器的耐久每10秒减少一次，耐久为0时失去生铸特性，变回原装备。

- 战斗结束后，所有生铸神器都将失去生铸特性，变回原装备。

<span style="color:#9eb767">[架海神杖</span>](../../../词条/功法/架海神杖.md)的生铸效果仅为敌人的装备添加生铸特性，不会将对应装备变为生铸神器。
