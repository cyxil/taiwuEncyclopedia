# 部位状态

> 百科 Key: `战斗-标记与状态-部位状态`
> 层级: 战斗 / 标记与状态 / 部位状态

在战斗中，人物的各个身体部位可能会出现各种负面状态。


### 破绽

当人物使用<span style="color:#6fb6ff">[变招](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#变招)</span>或<span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>命中敌人后，有可能使敌人的对应身体部位出现破绽，并产生<span style="color:#6fb6ff">[破绽标记](../战败标记/战斗-标记与状态-战败标记.md#破绽标记)</span>。

出现破绽的部位受到的直接伤害提高40%，同部位的多个破绽效果不会叠加。


破绽分为1~3三个等级。

- 以不同兵器变招造成的破绽等级可能不同。

- 摧破功法的品级越高，攻击造成的破绽等级越高。<span style="color:#6fb6ff">[下三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[中三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[上三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>功法造成的破绽分别为1/2/3级。

- 某些功法的效果可能将破绽等级提升至4级。


每个破绽会在出现一段时间后消失。等级越高的破绽消失得越慢。

每个身体部位最多存在3个破绽。当破绽数量达到上限后，新出现的破绽会替换等级更低的破绽。

战斗结束后，人物身上的所有破绽将会移除。


### 封穴

当人物使用<span style="color:#6fb6ff">[变招](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#变招)</span>或<span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>命中敌人后，有可能使敌人的攻击部位出现封穴，并产生<span style="color:#6fb6ff">[封穴标记](../战败标记/战斗-标记与状态-战败标记.md#封穴标记)</span>。

不同部位的封穴将对人物造成不同的妨害效果，同部位的多个封穴效果不会叠加。

当一个部位存在3个封穴时，人物将无法施展需要此部位的功法。

**各部位封穴效果一览**

| 部位 | 封穴效果 |
| --- | --- | 
| 头颈 | 头颈封穴会减少人物至少0.4、至多0.8的<span style="color:#6fb6ff">[攻击范围](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#攻击范围)</span> |
| 胸背 | 胸背封穴会使人物<span style="color:#6fb6ff">[提气](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md#提气)</span>的恢复速度降低至少20%、至多60% |
| 腰腹 | 腰腹封穴会使人物<span style="color:#6fb6ff">[架势](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md#架势)</span>的恢复速度降低至少20%、至多60% |
| 左臂 | 左臂封穴会使人物进行兵器攻击的<span style="color:#6fb6ff">[攻击准备](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#攻击耗时)</span>时间提高至少50%、至多150% |
| 右臂 | 右臂封穴会使人物进行兵器攻击的<span style="color:#6fb6ff">[攻击准备](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#攻击耗时)</span>时间提高至少50%、至多150% |
| 左腿 | 左腿封穴会使人物进行<span style="color:#6fb6ff">[移动](../../距离与移动/移动/战斗-距离与移动-移动.md)</span>所需的时间提高至少50%、至多150% |
| 右腿 | 右腿封穴会使人物进行<span style="color:#6fb6ff">[移动](../../距离与移动/移动/战斗-距离与移动-移动.md)</span>所需的时间提高至少50%、至多150% |


封穴分为1~3三个等级。

- 以不同兵器变招造成的封穴等级可能不同。

- 摧破功法的品级越高，攻击造成的封穴等级越高。<span style="color:#6fb6ff">[下三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[中三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>/<span style="color:#6fb6ff">[上三品](../../../物品/物品与资源/物品/物品-物品与资源-物品.md#品级)</span>功法造成的封穴分别为1/2/3级。

- 某些功法的效果可能将封穴等级提升至4级。


每个封穴会在出现一段时间后消失。等级越高的封穴消失得越慢。

每个身体部位最多存在3个封穴。当封穴数量达到上限后，新出现的封穴会替换等级更低的封穴。

战斗结束后，人物身上的所有封穴将会移除。


### 重伤

当一个身体部位的内伤或外伤达到5层时，此部位处于重伤状态。

当人物施展功法时，若<span style="color:#6fb6ff">[施展部位](../../../修习/武学/功法/修习-武学-功法.md#施展需要)</span>处于重伤状态，则此功法的<span style="color:#6fb6ff">[施展时间](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md#施展时间)</span>加倍。


### 残毁

当一个身体部位的内伤或外伤达到6层时，此部位处于残毁状态。

人物无法施展需要残毁部位的功法。


### 剧痛与伤坏

剧痛或伤坏的效果以人物特性的方式呈现。
