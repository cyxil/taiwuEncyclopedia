# 施展功法

> 百科 Key: `战斗-战斗基础-施展功法`
> 层级: 战斗 / 战斗基础 / 施展功法

功法可以在战斗中为人物提供大量的助益。

只有人物已<span style="color:#6fb6ff">[运功](../../../修习/武学/运功/修习-武学-运功.md)</span>的功法才能在战斗中施展或生效。


各类型功法在战斗中的作用：

- <span style="color:#6fb6ff">[内功](../../../修习/武学/功法/修习-武学-功法.md)</span>、<span style="color:#6fb6ff">[奇窍功法](../../../修习/武学/功法/修习-武学-功法.md)</span>：无需施展即可生效。可以在战斗双方的状态栏旁查看当前生效中的内功和奇窍功法。

- <span style="color:#6fb6ff">[身法](../../距离与移动/身法/战斗-距离与移动-身法.md)</span>（轻灵功法）：需要消耗脚力施展，能够帮助人物移动。

- <span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>：需要消耗提气、架势和蓄式施展，能够攻击敌人。

  腿法不消耗蓄式，但需要消耗脚力。

  暗器、奇门（符咒）、御射类功法还需要消耗兵器的耐久。

- <span style="color:#6fb6ff">[护体功法](../../攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>：需要消耗提气、架势施展，能够防御敌人的攻击。


### 施展时间

功法的施展并非瞬间完成，而是需要经过一定的施展时间。

在功法施展期间，若无特殊效果说明，人物无法<span style="color:#6fb6ff">[移动](../../距离与移动/移动/战斗-距离与移动-移动.md)</span>、进行<span style="color:#6fb6ff">[兵器攻击](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md)</span>或各种<span style="color:#6fb6ff">[战斗行为](../../指令与行为/战斗行为/战斗-指令与行为-战斗行为.md)</span>。

人物可以主动中断施展中的功法。


不同功法所需的施展时间不同。

人物的<span style="color:#6fb6ff">[施展速度](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#次要属性)</span>越快，功法的施展时间越短。

<span style="color:#6fb6ff">[助战指令](../../指令与行为/助战指令/战斗-指令与行为-助战指令.md)</span>「催发」可以增加功法的施展进度。


施展时间计算公式：

施展时间（帧） = 功法所需施展进度 ÷ 每帧增加施展进度 × （100% + 功法效果影响）

- 每帧增加施展进度 = 60 + 施展速度 × 30%

- 每60帧为1秒


### 提气

提气用于在战斗中施展<span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>或<span style="color:#6fb6ff">[护体功法](../../攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>。

在战斗中，提气会随时间逐渐恢复。

人物的<span style="color:#6fb6ff">[提气恢复](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#次要属性)</span>越高，提气的恢复速度越快。

功法的<span style="color:#6fb6ff">[内功比例](../../../修习/武学/功法/修习-武学-功法.md#内外功比例)</span>越高，施展需要的提气越多。

提气恢复公式：

每帧恢复提气% = （30 + 30 × 提气恢复% ÷ 4） ÷ 30000

- 每60帧为1秒


### 架势

架势用于在战斗中施展<span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>或<span style="color:#6fb6ff">[护体功法](../../攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>。

当人物进行<span style="color:#6fb6ff">[兵器攻击](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md)</span>，或受到敌人的兵器攻击时，可以获得架势。

不同兵器每次攻击获得的架势不同。

攻击被<span style="color:#6fb6ff">[化解](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#命中与化解)</span>时也可获得架势；攻击<span style="color:#6fb6ff">[落空](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#攻击范围)</span>时不会获得架势。

人物的<span style="color:#6fb6ff">[架势恢复](../../../人物/属性/战斗属性/人物-属性-战斗属性.md#次要属性)</span>可以提高每次攻击获得的架势。

功法的<span style="color:#6fb6ff">[外功比例](../../../修习/武学/功法/修习-武学-功法.md#内外功比例)</span>越高，施展需要的架势越多。


架势获取公式：

架势获取% = （60 + 兵器的架势获取 ÷ 获取方式系数 × 架势恢复影响） ÷ 4000

- 获取方式系数：非追击 = 1；追击 = 4；受到攻击 = 3

- 架势恢复影响 = （100 + 架势恢复 ÷ 除数）%

  如果本次攻击不为追击，除数 = 4

  如果本次攻击为追击，则除数取决于兵器变招需要消耗的变招次数：一变招次数 = 6；二变招次数 = 4；三变招次数 = 2


### 脚力

脚力用于在战斗中施展<span style="color:#6fb6ff">[身法](../../距离与移动/身法/战斗-距离与移动-身法.md)</span>或<span style="color:#6fb6ff">[腿法](../../../修习/武学/功法/修习-武学-功法.md)</span>。

在身法持续期间，人物的脚力将会持续消耗。如果人物在此期间<span style="color:#6fb6ff">[移动](../../距离与移动/移动/战斗-距离与移动-移动.md)</span>，还会加快脚力的消耗。消耗的速度取决于身法的施展效果。

当脚力降低至0%时，持续的身法将会中断。

不在身法持续期间时，人物移动不会消耗脚力，且脚力会随时间逐渐恢复（每秒恢复10%）。


### 蓄式

蓄式用于在战斗中施展<span style="color:#6fb6ff">[摧破功法](../../攻击与防御/摧破功法/战斗-攻击与防御-摧破功法.md)</span>，主要通过<span style="color:#6fb6ff">[兵器攻击](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md)</span>获取。


每当兵器攻击命中后，根据此次攻击的<span style="color:#6fb6ff">[招式](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#招式)</span>，人物将获得1个对应的蓄式。

如果兵器攻击被<span style="color:#6fb6ff">[化解](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#命中与化解)</span>，仍有一定概率获得蓄式。

兵器攻击的每次<span style="color:#6fb6ff">[追击](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#追击)</span>均有概率获得蓄式。

人物最多可以拥有9个蓄式。当蓄式数量达到上限后，新获得的蓄式会代替最早获得的蓄式。


与人物当前使用兵器的<span style="color:#6fb6ff">[招式](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md#招式)</span>类型对应的蓄式，称为「可用蓄式」。只有可用蓄式能够用于摧破功法的<span style="color:#6fb6ff">[施展需要](../../../修习/武学/功法/修习-武学-功法.md#施展需要)</span>。

如果当前兵器的招式未包含摧破功法需要的所有蓄式，则必定无法施展此摧破功法。


除了通过兵器攻击获得蓄式外，部分功法的心法效果也能使人物获得蓄式。

某些功法的心法效果还能使人物获得特殊的蓄式：

- 「杀」式：通过<span style="color:#6fb6ff">[界青门](../../../门派/门派一览/界青门/门派-门派一览-界青门.md)</span>功法获得的特殊蓄式，可以为界青门功法的效果提供加成。

- 「无」式：部分功法能够使敌人获得的特殊蓄式，主要用于扰乱敌人的蓄式，或触发某些特殊效果。

这些特殊蓄式均非可用蓄式，且同样计入人物的蓄式数量上限。


兵器攻击被化解时，获得蓄式概率 = 10% + <span style="color:#6fb6ff">[命中几率](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#命中几率)</span> ÷ 2

如果攻击被「必然化解」的效果化解，则不会获得蓄式。


追击获得蓄式概率：

在同一次兵器攻击中，追击获得蓄式的概率初始为20%，每次判定失败后+20%，判定成功后重置为20%。
