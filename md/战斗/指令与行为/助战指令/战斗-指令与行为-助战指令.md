# 助战指令

> 百科 Key: `战斗-指令与行为-助战指令`
> 层级: 战斗 / 指令与行为 / 助战指令

在战斗中，人物可以通过助战指令，指挥各自的<span style="color:#6fb6ff">[助战同道](../../../交互/其他交互/同道/交互-其他交互-同道.md#助战同道)</span>协同作战。


每名人物最多拥有3个助战指令。

月份更替时，人物的助战指令会根据其各项<span style="color:#6fb6ff">[特性倾向](../../../人物/属性/特性/人物-属性-特性.md#特性倾向)</span>的数量发生变化。

太吾可以为自己的同道更换助战指令。更换后，同道的助战指令将不再随月份更替而变化。

更换助战指令时，各项指令需要的特性倾向之和不能超过人物拥有的特性倾向。

完成狮相门的地区故事后，可以通过<span style="color:#6fb6ff">[统筹方略](../../../门派/门派一览/狮相门/门派-门派一览-狮相门.md#特有功能)</span>，强化同道的助战指令。

在战斗中使用助战指令后，此指令需要经过一段冷却时间才能再次使用。不同助战指令的冷却时间不同。助战同道对主战者的好感越高，助战指令的冷却时间越短。

如果助战同道对主战者的好感达到一定程度，其助战指令每次使用后，将有25%的概率跳过冷却，可以直接再次使用。


助战指令冷却时间计算公式：

冷却时间（帧） = 指令冷却所需进度 ÷ 每帧增加进度

- 每帧增加进度：由同道对主战者的好感决定。

  <span style="color:#C6272EFF">憎恨</span>及以下 = 10；达到<span style="color:#C6272EFF">敌视</span> = 25；达到<span style="color:#B6B1ABFF">陌路</span> = 40；达到<span style="color:#239360FF">冷淡</span> = 70；达到<span style="color:#1ABAC8FF">热忱</span> = 85；达到<span style="color:#E9D382FF">亲密</span> = 100

- 每60帧为1秒


### 负面指令

战斗开始时，若助战同道对主战者的好感为<span style="color:#C6272EFF">鄙视</span>及以下，助战指令会被随机替换为负面指令。

在战斗中，助战同道会自动随机使用负面指令，对己方主战者造成不利影响。

若好感为<span style="color:#C6272EFF">鄙视</span>或<span style="color:#C6272EFF">敌视</span>，出现1个负面助战指令；若好感为<span style="color:#C6272EFF">仇视</span>或<span style="color:#C6272EFF">憎恨</span>，出现2个负面助战指令；若好感为<span style="color:#C6272EFF">痛恨</span>或<span style="color:#C6272EFF">血仇</span>，出现3个负面助战指令。

**负面指令一览**

| 助战指令 | 特性倾向 | 效果 |
| --- | --- | --- | 
| 打断 | 守御 | 因心生嫌隙，可能在主战者<span style="color:#6fb6ff">[施展功法](../../战斗基础/施展功法/战斗-战斗基础-施展功法.md)</span>时，伺机将主战者的施展打断。 |
| 陷害 | 机略 | 因心生嫌隙，可能在主战者不在敌人的<span style="color:#6fb6ff">[攻击范围](../../攻击与防御/攻击/战斗-攻击与防御-攻击.md#攻击范围)</span>内时，伺机将主战者向敌人的攻击范围内推拉。 |
| 偷袭 | 进攻 | 因心生嫌隙，可能伺机偷袭主战者，对主战者进行<span style="color:#6fb6ff">[兵器攻击](../../攻击与防御/兵器攻击/战斗-攻击与防御-兵器攻击.md)</span>，并造成<span style="color:#6fb6ff">[破绽](../../标记与状态/部位状态/战斗-标记与状态-部位状态.md#破绽)</span>、<span style="color:#6fb6ff">[封穴](../../标记与状态/部位状态/战斗-标记与状态-部位状态.md#封穴)</span>。 |
| 散功 | 守御 | 因心生嫌隙，可能伺机消除主战者持续中的<span style="color:#6fb6ff">[护体功法](../../攻击与防御/护体功法/战斗-攻击与防御-护体功法.md)</span>、<span style="color:#6fb6ff">[身法](../../距离与移动/身法/战斗-距离与移动-身法.md)</span>。 |
| 暗害 | 进攻 | 因心生嫌隙，可能伺机以医术、毒术对主战者造成伤势、毒素。 |
| 纠缠 | 进攻 | 因心生嫌隙，可能伺机在短时间内，降低主战者的所有命中属性和化解属性。 |
| 扰乱 | 守御 | 因心生嫌隙，可能在主战者进行各项<span style="color:#6fb6ff">[战斗行为](../战斗行为/战斗-指令与行为-战斗行为.md)</span>时，伺机将行为打断。 |
| 散气 | 机略 | 因心生嫌隙，可能伺机消散主战者的真气。 |
