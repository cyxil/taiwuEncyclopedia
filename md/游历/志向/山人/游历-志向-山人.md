# 山人

> 百科 Key: `游历-志向-山人`
> 层级: 游历 / 志向 / 山人

*心有长林丰草，甘为山林之士，清苦鄙陋何足忧，且为人间蓬蒿客……*


志向造诣：道法

志向衣装：<span style="color:#9eb767">[山野竖褐</span>](../../../词条/衣装/山野竖褐.md)


### 资历获取

山人志向资历获取方式：

- 地格互动：<span style="color:#6fb6ff">[采集](../../../世界/地图互动/地格互动/世界-地图互动-地格互动.md#采集)</span>、<span style="color:#6fb6ff">[挖掘](../../../世界/地图互动/地格互动/世界-地图互动-地格互动.md#挖掘)</span>

- 物品：<span style="color:#6fb6ff">[拆解](../../../物品/制造加工/拆解/物品-制造加工-拆解.md)</span>引子、养育<span style="color:#6fb6ff">[神木](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#神木种子)</span>

- 奇遇：在「天材地宝」<span style="color:#6fb6ff">[奇遇](../../../世界/地图互动/奇遇/世界-地图互动-奇遇.md)</span>中获得引子


山人志向资历获取公式：

- 于地格主动采集时：

  增长资历 = 采集前的地格资源量 × 20

  采集到引子时：增长资历 = 采集所得引子的价值

  采集到神木种子时：增长资历 = 28200

- 挖掘获得物品时：增长资历 = 所得物品的价值

- 拆解引子时：增长资历 = 拆解引子的价值 ÷ 2

- 养育神木时：增长资历 = 将神木种子养育成神木时，获得123000资历

- 于天材地宝奇遇中获得引子时：增长资历 = 所得引子的价值 × 250%


### 山水难阻

技能等级：一级

解锁节点：山人资历达到5%


使用方式：被动生效

技能效果：增加<span style="color:#6fb6ff">[膂力](../../../人物/属性/主要属性/人物-属性-主要属性.md)</span>，并提高月份更替时膂力的恢复量。

- 山人资历越高，增加的膂力、提高的膂力恢复量越多。

效果公式：

增加膂力 = 10 + 20 × 山人资历百分比

提高膂力恢复量% = 100 × 山人资历百分比


### 休养生息

技能等级：二级

解锁节点：山人资历达到20%


使用方式：主动使用

使用消耗：10精力、500历练

使用冷却：3个月

技能效果：恢复1格范围内的所有<span style="color:#6fb6ff">[地格资源](../../../世界/地图/地格/世界-地图-地格.md#地格资源)</span>后，同时在这些地格上<span style="color:#6fb6ff">[采集](../../../世界/地图互动/地格互动/世界-地图互动-地格互动.md#采集)</span>所有类型的资源。本次采集不会消耗地格资源。

- 山人资历越高，地格资源的恢复比例越高。

- 如果1格范围内有种植的<span style="color:#6fb6ff">[神木种子](../../../物品/物品类型/杂物/物品-物品类型-杂物.md#神木种子)</span>，还会提升其1个成长阶段。

效果公式：

恢复的地格资源量 = 该地格对应资源上限 × （33 + 33 × 山人资历百分比）%


### 因地制宜

技能等级：三级

解锁节点：山人资历达到50%


使用方式：被动生效

技能效果：战斗开始时，根据1格范围内的所有<span style="color:#6fb6ff">[野外地形](../../../世界/地图/地格/世界-地图-地格.md#地形)</span>，获得相应的加成效果。

- 多个地形的同类效果不会叠加。

**因地制宜效果一览**

| 效果名称 | 地形 | 效果 |

| 凌绝顶 | 山岳、山脉 | 进行兵器攻击耗时降低20% |
| 一线天 | 峡谷、天险 | 功法施展速度提高20% |
| 九折径 | 丘陵、高地 | 敌人的命中几率降低20% |
| 苍茫野 | 原野、平原 | 我方的命中几率提高20% |
| 连山翠 | 林地、森林 | 提气、架势恢复提高20% |
| 空行涧 | 河滩、河谷、溪谷 | 脚力恢复速度提高20% |
| 烟波荡 | 湖泊 | 身法蓄力速度提高20% |
| 森罗嶂 | 密林 | 变招获取速度提高20% |
| 岩穴暝 | 洞穴 | 敌人兵器攻击耗时提高20% |
| 幽潭沉 | 沼泽 | 战斗中对敌人施加的毒素量提高50%，毒素等级提升一级 |
| 桃花源 | 桃源 | 战斗结束后恢复所有消耗的内力 |


### 山野遗宝

技能等级：四级

解锁节点：山人资历达到100%


使用方式：主动使用

使用消耗：10精力、5000历练

使用冷却：3个月

技能效果：指定并获得当前地格可<span style="color:#6fb6ff">[挖掘](../../../世界/地图互动/地格互动/世界-地图互动-地格互动.md#挖掘)</span>的任意一件物品。
